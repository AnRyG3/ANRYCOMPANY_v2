# 台本品質評価システム

## 目的

台本を「とりあえず完成」ではなく、「伸びる可能性が高い状態」までブラッシュアップする。

AIが自己採点し、改善案を出し、120点に近づくまで仮想改善を繰り返す。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| フック | /10 | 3秒以内に見たいと思えるか |
| 分かりやすさ | /10 | 一般人に伝わる言葉か |
| 驚き | /10 | “え？”と思う要素があるか |
| 保存価値 | /10 | 後で見返したくなるか |
| 共感 | /10 | 不安や経験に寄り添えているか |
| 専門性 | /10 | 放射線技師視点があるか |
| 安心感 | /10 | 最後に不安が下がるか |
| テンポ | /10 | 1文が短く流れがよいか |
| コメントしたくなるか | /10 | 問いかけが自然か |
| 最後まで見たくなるか | /10 | 展開に引きがあるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 作り直し | フック、結論、構成を全面修正 |
| 90点以上 | 投稿可能 | 制作に進める |
| 100点 | 強い | 優先的に投稿 |
| 120点 | バズ候補 | 最高評価。疑似加点で狙う理想値 |

## 120点評価について

基本採点は100点満点。

ただし、以下のボーナス要素が強い場合は最大20点を加点し、120点満点として扱う。

| ボーナス | 最大点 |
|---|---:|
| 最初の1秒が強い | +5 |
| 保存したくなる実用性が高い | +5 |
| 放射線技師だから言える現場感がある | +5 |
| コメントが生まれやすい問いがある | +5 |

## 改善フロー

1. 初稿を作る
2. 10項目で採点する
3. 80点未満なら作り直す
4. 90点以上なら投稿可能
5. ボーナス加点を狙って改善する
6. 120点に近づくまで、フック・保存価値・技師視点・安心感を磨く
