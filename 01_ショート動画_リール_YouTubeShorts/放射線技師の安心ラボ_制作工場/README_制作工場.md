# 放射線技師の安心ラボ Instagramリール制作工場

## 目的

Instagramリールを週3〜5本、毎回ゼロから考えずに量産するための制作システム。

この制作工場の目的は、医療知識を一般人向けにわかりやすく伝え、放射線への誤解を解き、不安を安心に変える保存されるリールを継続的に作ること。

## 全体フロー

1. `01_ネタ` でテーマを選ぶ
2. `02_台本` で台本を作る
3. 台本品質評価システムで自己採点する
4. 90点以上なら制作へ進む
5. 120点に近づくまで仮想改善する
6. `03_画像` で画像プロンプトを作る
7. `04_音声` でナレーション音声を作る
8. `05_Vrew` で字幕下地を作る
9. `06_CapCut` で編集する
10. `07_投稿済` に完成物と投稿文を保存する
11. `08_分析` で結果を記録し、次回へ反映する

## 量産ルール

- 3秒で興味を引く
- 結論を先に出す
- 難しい言葉を使わない
- 一般人の誤解を扱う
- 放射線技師視点を入れる
- 最後は安心感で終わる
- 保存誘導を入れる
- コメント誘導を入れる
- 医療不安を煽りすぎない

## 週3〜5本投稿の型

| 曜日 | 投稿内容 |
|---|---|
| 月曜 | 放射線の誤解解消 |
| 火曜 | レントゲン豆知識 |
| 水曜 | CT・MRIの不安解消 |
| 木曜 | 技師あるある |
| 金曜 | 整形撮影法・医療雑学 |

週3本の場合は、月曜・水曜・金曜だけ運用する。
