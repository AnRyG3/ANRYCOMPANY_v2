# 画像プロンプトテンプレ

## Gemini用プロンプト

```text
Instagramリール用の医療系ビジュアルを作成してください。

テーマ:
{テーマ}

アカウント:
放射線技師の安心ラボ

条件:
- 日本人向け
- 医療系
- Instagram映え
- 9:16比率
- 白背景ベース
- 青・水色系
- 清潔感
- シンプル
- 一般人にも怖く見えない
- 不安を煽らない
- 文字は入れない
- CanvaやCapCutで文字を入れやすい余白を残す
- 背景や図解自体が説明している場合は、その意味が隠れない構図にする
- 1画面1メッセージで、視線が散る装飾やラベルを増やしすぎない

入れたい要素:
- 放射線技師または医療スタッフ
- 患者さんが安心して検査を受ける雰囲気
- レントゲン、CT、MRI、検査室などテーマに合う医療要素
- やさしい表情
- 信頼感のある構図

避ける表現:
- 怖い放射線表現
- 赤色の警告だらけ
- 不安そうな患者さんだけを強調する構図
- リアルすぎて怖い医療表現
```

## 整形撮影法用

```text
Instagramリール用に、整形外科X線撮影法の説明イラストを作成してください。

撮影法:
{撮影法名}

条件:
- 9:16比率
- 白背景
- 青・水色系
- 医療系の清潔感
- シンプルな図解風
- 患者さんの体位が分かる
- 関節の向きや角度が分かる
- 文字は入れない
- 不安を煽らない
- 一般人にも医療従事者にも見やすい
- 体位や角度など、画像で伝えるべき要素をテロップ予定位置で隠さない
```
