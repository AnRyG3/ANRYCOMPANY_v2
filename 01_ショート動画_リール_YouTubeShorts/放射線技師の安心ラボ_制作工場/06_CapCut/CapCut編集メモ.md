# CapCut編集メモ

## 基本設定

| 項目 | 設定 |
|---|---|
| 比率 | 9:16 |
| 解像度 | 1080x1920 |
| 尺 | 15〜35秒 |
| 書き出し | MP4 |

## テロップルール

- 文字は大きく
- 1画面最大2行
- 1行12〜16文字程度
- 白背景に濃いグレー文字
- 重要語だけ青・水色で強調
- 注意はやわらかい赤かオレンジ
- 専門用語はすぐ言い換える
- 3秒で内容が分かるようにする
- 1画面1メッセージにする
- 背景や図解が説明している時は、テロップで大事な情報を隠さない
- 「温泉の雑学」「安心ポイント」などのカテゴリラベルは、必要な時だけ使う
- テロップは短く強く、画像は説明、文字は補助として使う

## カットテンポ

- 冒頭3秒は1秒ごとに画面変化
- 通常部分は2〜3秒ごとに変化
- 同じ画面を長く続けない
- 図解、ズーム、テロップ強調で変化を出す
- 最後は少し落ち着かせて安心感を出す

## BGMルール

- ナレーションを邪魔しない
- 音量は小さめ
- 清潔感、安心感、やさしさのあるBGM
- 不安を煽るBGMは使わない
- 警告音のような音は避ける

## SEルール

- 冒頭のフックに軽いSEを1つ
- 重要語の表示に短いSE
- 多用しない
- 医療情報の信頼感を壊さない

## 編集後チェック

- [ ] 冒頭1秒で止まる要素がある
- [ ] 3秒以内にテーマが分かる
- [ ] 結論が早い
- [ ] テロップが読める
- [ ] テロップが背景や図解の重要部分を隠していない
- [ ] 不要なカテゴリラベルで視線が散っていない
- [ ] BGMがうるさくない
- [ ] 最後が安心感で終わる
- [ ] 保存誘導が入っている
- [ ] コメント誘導が入っている
