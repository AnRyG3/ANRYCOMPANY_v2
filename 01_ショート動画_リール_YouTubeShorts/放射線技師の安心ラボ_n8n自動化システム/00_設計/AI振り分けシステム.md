# AI振り分けシステム

## 目的

用途ごとにAIを分岐し、品質とコストを最適化する。

## AI別役割

| AI | 得意用途 | 使用場面 |
|---|---|---|
| ChatGPT | 構成、分析、SEO、台本、テンプレ化 | ネタ生成、台本、投稿分析、SEO |
| Claude | 長文、深掘り、自然な文章 | ブログ本文、長文リライト |
| Gemini | 柔らかい表現、画像プロンプト、一般向け表現 | 一般向け言い換え、画像生成指示 |
| ImageFX | 画像生成 | サムネ背景、イラスト素材 |
| Vrew | 字幕・音声下地 | ナレーション字幕 |
| Canva | デザイン | サムネ、図解、文字入れ |
| CapCut | 動画編集 | リール完成 |

## n8n分岐ルール

```text
if task = "台本" → ChatGPT
if task = "ブログ長文" → Claude or ChatGPT
if task = "SEO" → ChatGPT
if task = "画像プロンプト" → Gemini
if task = "投稿分析" → ChatGPT
if task = "LINE文言" → ChatGPT
if task = "柔らかい女性向け表現" → Gemini
```

## コストを抑えるルール

- 初稿は低コストAI
- 重要投稿だけ高性能モデル
- 画像生成は候補を絞ってから行う
- 長文ブログは見出し確定後に生成する
- 分析は週次まとめで一括実行する
