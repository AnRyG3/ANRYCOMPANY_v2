# Google Spreadsheet設計

## シート構成

```text
01_ネタ管理
02_リール管理
03_ブログ管理
04_LINEスタンプ管理
05_投稿分析
06_成功パターン
07_エラー管理
```

## 01_ネタ管理

| 項目 | 内容 |
|---|---|
| ID | 自動採番 |
| タイトル | 投稿タイトル |
| ジャンル | CT、MRI、レントゲンなど |
| ターゲット | 一般人、医療者、若手技師 |
| フック | 冒頭3秒 |
| 投稿状態 | 未作成、台本済、編集中、投稿済 |
| 保存率 | 投稿後に記録 |
| 再生数 | 投稿後に記録 |
| 投稿日 | 投稿予定日または投稿日 |
| メモ | 改善点 |

## 02_リール管理

| 項目 | 内容 |
|---|---|
| Reel ID | 投稿ID |
| テーマ | 動画テーマ |
| 台本 | 台本URL |
| 画像プロンプト | prompt URL |
| 音声 | Drive URL |
| 動画 | Drive URL |
| サムネ | Drive URL |
| 説明文 | caption |
| ハッシュタグ | hashtags |
| 投稿状態 | 未投稿、予約済、投稿済 |

## 03_ブログ管理

| 項目 | 内容 |
|---|---|
| 記事ID | 自動採番 |
| 記事名 | ブログタイトル |
| 撮影法名 | ローゼンバーグなど |
| SEOタイトル | 検索向けタイトル |
| メタディスクリプション | SEO説明文 |
| 画像 | Drive URL |
| Google Docs URL | 下書き |
| 公開URL | 公開後 |
| 状態 | 下書き、確認中、公開済 |

## 04_LINEスタンプ管理

| 項目 | 内容 |
|---|---|
| スタンプID | 自動採番 |
| テーマ | 日常、推し活、敬語など |
| タイトル | 販売タイトル |
| 40個文言 | 文言一覧URL |
| 画像プロンプト | prompt URL |
| 販売状態 | 未作成、制作中、販売中 |
| 審査状態 | 未申請、審査中、承認、リジェクト |
| メモ | 修正点 |

## 05_投稿分析

| 項目 | 内容 |
|---|---|
| 投稿ID | Reel ID |
| 投稿日時 | 投稿時間 |
| 再生数 | 数値 |
| 保存数 | 数値 |
| コメント数 | 数値 |
| フォロー増加 | 数値 |
| 完視聴率 | 数値 |
| ハッシュタグ | 使用タグ |
| 成功判定 | A、B、C |

## 06_成功パターン

| 項目 | 内容 |
|---|---|
| パターンID | 自動採番 |
| 元投稿ID | 参照ID |
| 成功理由 | 保存、再生、フォローなど |
| フック型 | 誤解、比較、不安解消 |
| 構成 | 0〜35秒構成 |
| 次回派生案 | シリーズ案 |

## 07_エラー管理

| 項目 | 内容 |
|---|---|

| エラー内容 | API失敗など |
| 対応状況 | 未対応、対応済 |
| メモ | 原因 |
