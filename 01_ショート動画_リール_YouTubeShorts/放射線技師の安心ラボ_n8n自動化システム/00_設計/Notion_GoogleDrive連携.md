# Notion・Google Drive連携

## Notion連携

### 自動保存するもの

- ネタDB
- 投稿分析
- バズ構成
- ブログ一覧
- 改善案
- 成功パターン

### Notion DB構成

```text
ネタDB
投稿分析DB
バズ構成DB
ブログ管理DB
LINEスタンプ管理DB
改善案DB
```

```text
Google Sheets Trigger

```

## Google Drive連携

### 自動整理フォルダ

```text
放射線技師の安心ラボ/
├─ 01_ネタ/
├─ 02_台本/
├─ 03_画像/
├─ 04_音声/
├─ 05_動画/
├─ 06_ブログ/
├─ 07_サムネ/
├─ 08_投稿予約/
├─ 09_分析/
└─ 10_バックアップ/
```

### 保存ルール

| データ | 保存先 |
|---|---|
| 台本 | 02_台本 |
| 画像プロンプト | 03_画像 |
| 音声台本 | 04_音声 |
| 完成動画 | 05_動画 |
| ブログ下書き | 06_ブログ |
| サムネ案 | 07_サムネ |
| 投稿文 | 08_投稿予約 |
| 分析レポート | 09_分析 |
| 週次バックアップ | 10_バックアップ |
