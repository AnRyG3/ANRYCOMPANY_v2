# Instagramリール動画制作 完全テンプレート

## 目的

医療系アカウント「放射線技師の安心ラボ」のInstagramリール動画を効率よく量産する。

放射線や検査に不安を持つ一般の人へ、やさしく、わかりやすく、安心できる医療情報を届ける。

## アカウント設計

| 項目 | 内容 |
|---|---|
| アカウント名 | 放射線技師の安心ラボ |
| 発信テーマ | 放射線検査、レントゲン、CT、医療雑学、整形撮影法 |
| 基本姿勢 | 怖がらせず、正しく知って安心してもらう |
| ターゲット | 一般の人、放射線が怖い人、医療知識が少ない人、20〜50代 |
| トーン | やさしい、専門的すぎない、安心感がある |
| 投稿尺 | 15〜35秒 |
| 文章 | 1文を短く、テンポ重視 |

## CapCut編集テンプレ

## 基本設定

| 項目 | 設定 |
|---|---|
| 画面比率 | 9:16 |
| 解像度 | 1080x1920 |
| 尺 | 15〜35秒 |
| フレーム | 30fps |
| 書き出し | MP4 |
| 音声 | ナレーション中心 |

## 編集構成

```text
0〜1秒: 強いサムネ文字をそのまま表示
1〜3秒: フックを大きく表示
3〜8秒: 問題提起と結論
8〜20秒: 図解・比較・理由説明
20〜30秒: 安心感のあるまとめ
30〜35秒: 保存・フォロー誘導
```

## 編集ルール

- 1カットは1〜3秒を目安にする
- 画面の変化を3秒以内に入れる
- テロップは常に大きく表示する
- 重要語だけ色を変える
- 余計なエフェクトを入れすぎない
- 医療系なので落ち着いたテンポにする
- 最後は安心感のある静かな着地にする

## テロップルール

## 基本デザイン

- 白背景ベース
- 青・水色系をメインカラーにする
- 文字は大きく、太く、短く
- 1画面の文字は最大2行
- 1行は12〜16文字程度を目安にする
- 専門用語はなるべく使わない
- 使う場合はすぐに言い換える

## 色ルール

| 用途 | 色 |
|---|---|
| 通常文字 | 濃いグレー、黒 |
| 重要語 | 青、水色 |
| 注意 | やわらかい赤、オレンジ |
| 安心 | 水色、淡い緑 |
| 背景 | 白、薄い水色 |

## テロップ例

```text
CTって怖い？
結論、必要な検査なら大丈夫
被ばくは必要最小限
不安な時は相談してOK
```

## BGMルール

- ナレーションを邪魔しない音量にする
- BGM音量は小さめにする
- 医療系なので不安を煽る音は避ける
- 明るすぎるバラエティ系BGMは使いすぎない
- 清潔感、安心感、やさしさのある曲を選ぶ
- シリアスなテーマではBGMを薄くする
- 最後は安心して終われる雰囲気にする
