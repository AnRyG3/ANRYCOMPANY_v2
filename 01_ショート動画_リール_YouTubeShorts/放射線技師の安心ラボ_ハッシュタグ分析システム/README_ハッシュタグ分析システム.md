# 放射線技師の安心ラボ ハッシュタグ分析システム

## 目的

Instagramリール、TikTok、YouTube Shortsで「放射線技師の安心ラボ」の投稿ごとに最適なハッシュタグ戦略を作る。

目的は、再生数向上、保存率向上、発見タブ流入、フォロワー増加、医療系ターゲットへの到達、一般人への拡散。

## 対象テーマ

- 放射線
- レントゲン
- CT
- MRI
- 被ばく
- 医療雑学
- 安心解説
- 診療放射線技師
- 整形外科
- 医療系教育

## フォルダ構成

```text
放射線技師の安心ラボ_ハッシュタグ分析システム/
├─ 01_タグDB/
├─ 02_投稿別分析/
├─ 03_競合分析/
├─ 04_保存率分析/
├─ 05_ジャンル別/
├─ 06_バズ分析/
├─ 07_禁止タグ/
├─ 08_テンプレ/
└─ 09_更新履歴/
```

## 運用フロー

1. 投稿テーマを決める
2. `05_ジャンル別/投稿テーマ別タグDB.md` から近いテーマを選ぶ
3. `08_テンプレ/投稿自動最適化テンプレ.md` にテーマを入力する
4. 推奨タグ15個を作る
5. `07_禁止タグ/禁止制限タグ管理.md` で危険タグを確認する
6. 投稿後に `02_投稿別分析/投稿別タグ分析テンプレ.md` へ結果を記録する
7. 伸びたタグを `06_バズ分析/タグ効果DB.md` に反映する

## 基本方針

- 大規模タグだけに寄せない
- 中規模・小規模・ニッチタグを混ぜる
- 投稿内容と説明文に関連するタグだけ使う
- 医療不安を煽るタグは避ける
- 指名タグ `#放射線技師の安心ラボ` を継続使用する
- 保存率を狙う投稿では「保存推奨」「検査前」「医療知識」系を入れる
