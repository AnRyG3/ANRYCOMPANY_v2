# タグ禁止・制限分析

## 目的

シャドウバン疑惑、スパム判定、競争率過多、伸びないタグ、医療広告リスクを管理する。

## 管理表

| タグ | リスク分類 | 理由 | 対応 |
|---|---|---|---|
| #絶対安全 | 医療広告リスク | 医療情報として断定が強い | 使用しない |
| #危険 | 不安煽り | 放射線不安を煽りやすい | 文脈限定でも避ける |
| #治る | 医療広告リスク | 効果保証に見える | 使用しない |
| #診断 | 誤認表現 | 個別診断に見える可能性 | 慎重に使う |
| #病気確定 | 医療リスク | 診断断定に見える | 使用しない |
| #バズりたい | スパム判定 | 投稿品質と関係が薄い | 使用しない |
| #相互フォロー | スパム判定 | アカウント品質低下 | 使用しない |

## チェック項目

- [ ] 不安を煽るタグがない
- [ ] 医療効果を断定するタグがない
- [ ] 投稿内容と関係ないタグがない
- [ ] スパムっぽいタグがない
- [ ] 大規模タグだけに偏っていない
- [ ] 指名タグが入っている

## 注意

ハッシュタグの投稿数や制限状況は変化するため、月1回見直す。
