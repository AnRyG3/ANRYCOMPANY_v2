# バズ分析システム タグ効果DB

## 目的

どのタグがどの動画で効果があったかをDB化し、次回投稿のタグ選定に反映する。

## 分析項目

| 投稿ID | テーマ | 使用タグ | 再生数 | 保存率 | 離脱率 | コメント率 | フォロー率 | タグ一致率 | 評価 |
|---|---|---|---:|---:|---:|---:|---:|---:|---|
| 001 | CT被ばく | #CT検査 #被ばく #医療知識 |  |  |  |  |  |  |  |

## 計算式

```text
保存率 = 保存数 ÷ リーチ × 100
コメント率 = コメント数 ÷ リーチ × 100
フォロー率 = フォロー数 ÷ リーチ × 100
タグ一致率 = 投稿内容とタグが一致している数 ÷ 使用タグ数 × 100
```

## 評価基準

| 評価 | 条件 |
|---|---|
| A | 保存率・フォロー率が高い |
| B | 再生数は高いが保存率が普通 |
| C | タグと内容の一致が弱い |
| D | 次回使用しない |
