# ハッシュタグ品質評価システム

## 目的

投稿テーマに対して、最適なハッシュタグ戦略になっているかを採点し、120点に近づくまで改善する。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| 拡散力 | /10 | 大規模・中規模タグで広がるか |
| 専門性 | /10 | 放射線技師・医療職に届くか |
| 初心者到達率 | /10 | 一般人にも届くか |
| 保存率 | /10 | 保存したくなるタグがあるか |
| 検索性 | /10 | 検索キーワードと一致するか |
| 競合回避 | /10 | 大規模タグに偏っていないか |
| 発見欄適性 | /10 | 投稿内容とタグが一致しているか |
| コメント誘導 | /10 | 会話が生まれやすいか |
| ブランド化 | /10 | 指名タグが入っているか |
| 継続性 | /10 | 次回以降も比較分析できるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 改善 | タグの分類と関連性を見直す |
| 90点以上 | 投稿推奨 | 使用してよい |
| 100点 | 強い | 優先使用 |
| 120点 | バズ候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナスが強い場合、最大20点を加点して120点満点として扱う。

| ボーナス | 最大点 |
|---|---:|
| 投稿テーマとの一致率が非常に高い | +5 |
| 保存狙いタグが自然に入っている | +5 |
| 一般人と専門職の両方へ届く | +5 |
| 指名タグでブランド化できている | +5 |
