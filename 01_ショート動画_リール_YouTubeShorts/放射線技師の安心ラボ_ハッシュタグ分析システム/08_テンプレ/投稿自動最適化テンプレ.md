# 投稿自動最適化テンプレ

## 入力

```text
投稿テーマ:
ターゲット: 一般人 / 医療従事者 / 若手技師 / 初心者
ジャンル: レントゲン / CT / MRI / 被ばく / 医療雑学 / 整形撮影 / 技師あるある
狙い: 再生数 / 保存率 / フォロー率 / コメント
```

## 出力

```markdown

## 危険タグ


## 競合率

高 / 中 / 低

## 保存狙いタグ


## 初心者向けタグ


## 専門タグ


## 指名タグ

## 採用理由


```

## 自動選定ルール

- 一般向けなら `#医療知識` `#医療雑学` `#健康` を優先
- 検査不安系なら `#検査前` `#安心解説` `#保存推奨` を入れる
- CTなら `#CT検査` `#被ばく` `#放射線` を入れる
- MRIなら `#MRI検査` `#CT検査` で比較流入を狙う
- 整形撮影なら `#整形外科` `#X線撮影` `#ポジショニング` を入れる
- 医療職向けなら `#診療放射線技師` `#医療従事者` `#新人技師` を入れる
- 毎回 `#放射線技師の安心ラボ` を入れる
