# ショート動画・リール・YouTube Shorts プロジェクト

## このプロジェクトで扱うこと

- Instagramリール
- TikTok
- YouTube Shorts
- 放射線技師の安心ラボ
- SNS-OS
- ネタ生成
- 台本作成
- 画像生成
- 動画編集
- 投稿管理
- 保存率分析
- バズ分析
- n8n自動化

## チャット開始時の言い方

```text
今日はショート動画プロジェクトです。
```

## 主なフォルダ

- `放射線技師の安心ラボ_制作工場`
- `放射線技師の安心ラボ_n8n自動化システム`
- `SNS-OS`

## 作業ルール

- 投稿テーマごとに新規ファイルまたは新規フォルダで管理する
- 分析結果はSNS-OSに集約する
- 台本、画像、投稿文、分析は混ぜずに分ける
- リールは制作に入る前に、必ず台本をユーザーへ提出する
- 台本確認では、医療従事者目線と患者目線で食い違いが起きそうな表現を確認する
- ユーザーがOKを出した後に、画像、音声、動画、投稿文などの制作へ進む
- 医療表現は断定しすぎず、不安をあおらない形に整える

## 毎回の追加アウトプット

- メインがリール制作でも、Codexは毎回、LINEスタンプ案、ブログ記事案、横展開候補を出す
- 伸びたリールや制作中のテーマは、ブログ化、画像投稿化、LINEスタンプ販促導線へ展開する
- ユーザーは候補を見て、採用・修正・保留・却下を判断する
