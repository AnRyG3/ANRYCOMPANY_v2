# コメントされやすいネタ分析

## コメントが生まれる要素

| 要素 | 例 |
|---|---|
| 地域差 | みなさんの施設ではどうしていますか？ |
| 経験談 | 検査で怖かった経験ありますか？ |
| 知らなかった | 知らなかった人はコメントで教えてください |
| 怖かった経験 | CTとMRI、どちらが怖かったですか？ |
| 検査体験 | この検査を受けたことありますか？ |

## コメント誘導テンプレ

- あなたはどっちが不安ですか？
- この検査、受けたことありますか？
- 知らなかった人はコメントで教えてください。
- 医療職の方は、どう説明していますか？
- みなさんの施設ではどうしていますか？
