# NGネタ分析

## 回避するネタ

| NG | 理由 | 改善 |
|---|---|---|
| 不安煽り過ぎ | 医療不安を強める | 安心感で終わる |
| 専門用語過多 | 一般人が理解できない | 簡単な言葉にする |
| 炎上リスク | 誤解・批判につながる | 断定を避ける |
| 医療広告違反 | 効果保証に見える | 一般的な情報にする |
| 根拠不明 | 信頼低下 | 不確かな数字を避ける |
| 個別診断 | 医師の診断に見える | 相談を促す |

## チェックリスト

- [ ] 不安だけで終わっていない
- [ ] 絶対安全、絶対危険と言っていない
- [ ] 専門用語が多すぎない
- [ ] 個別診断に見えない
- [ ] 根拠不明の数字を使っていない
- [ ] 最後に安心感がある
