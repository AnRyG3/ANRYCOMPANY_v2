# ネタ品質評価システム

## 目的

ネタの段階で、バズ期待・保存価値・コメント期待・安心感・量産性を採点し、120点に近づける。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| バズ期待 | /10 | 再生されそうか |
| 保存価値 | /10 | 後で見返したいか |
| コメント期待 | /10 | 反応が生まれそうか |
| 安心感 | /10 | 不安を下げられるか |
| 一般人理解 | /10 | わかりやすいか |
| 専門性 | /10 | 技師視点があるか |
| シリーズ化 | /10 | 続編にできるか |
| 量産性 | /10 | 同じ型で増やせるか |
| トレンド性 | /10 | 季節や話題とつなげられるか |
| 再利用性 | /10 | ブログや画像投稿へ展開できるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 改善 | フック・保存価値・安心感を見直す |
| 90点以上 | 投稿推奨 | 制作へ進む |
| 100点 | 強い | 優先投稿 |
| 120点 | バズ候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナスが強い場合、最大20点を加点する。

| ボーナス | 最大点 |
|---|---:|
| 一般人が知らない要素が強い | +5 |
| 保存・コメント両方を狙える | +5 |
| シリーズ化しやすい | +5 |
| 複数媒体へ再利用できる | +5 |
