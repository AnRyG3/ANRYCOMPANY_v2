# ネタ使い回しシステム

## 目的

1つのネタを、リール・ブログ・ショート・画像投稿・LINE風投稿へ展開する。

## 展開テンプレ

| 展開先 | 内容 |
|---|---|
| リール | 15〜35秒の縦動画 |
| ブログ | 詳細解説記事 |
| YouTube Shorts | 検索寄りタイトルで投稿 |
| 画像投稿 | 3〜5枚の図解 |
| LINE風投稿 | 会話形式で不安解消 |

## 例

テーマ:
CTって危険？

| 展開先 | タイトル |
|---|---|
| リール | CTが怖い人へ |
| ブログ | CT被ばくは危険？放射線技師がわかりやすく解説 |
| Shorts | CTの被ばくはどれくらい？ |
| 画像投稿 | CT前に知っておきたい3つ |
| LINE風投稿 | 友達にCTが不安と相談されたら |
