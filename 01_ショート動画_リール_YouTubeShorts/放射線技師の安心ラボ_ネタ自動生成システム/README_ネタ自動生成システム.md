# 放射線技師の安心ラボ ネタ自動生成システム

## 目的

ネタ切れを防ぎ、放射線系コンテンツを継続的に量産するための制作工場。

一般人が興味を持ち、保存され、コメントされ、不安解消につながるネタを毎日作れる状態にする。

## 対象テーマ

- レントゲン
- CT
- MRI
- 放射線
- 被ばく
- 整形撮影
- 診療放射線技師
- 医療雑学
- 現場あるある
- 初心者向け医療知識

## フォルダ構成

```text
放射線技師の安心ラボ_ネタ自動生成システム/
├─ 01_ネタDB/
├─ 02_一般人向け/
├─ 03_医療者向け/
├─ 04_保存型/
├─ 05_驚き型/
├─ 06_不安解消型/
├─ 07_比較型/
├─ 08_シリーズ化/
├─ 09_トレンド連携/
└─ 10_投稿済管理/
```

## 運用フロー

1. ネタカテゴリDBから方向性を選ぶ
2. ネタ生成システムにテーマ・ターゲット・不安を入力する
3. タイトル、フック、構成、保存誘導、コメント誘導、サムネ案を出す
4. バズ予測で90点以上を選ぶ
5. シリーズ化できるネタはシリーズDBへ追加する
6. 投稿済管理で被りを防ぐ
7. 成功ネタはブログ、ショート、画像投稿へ使い回す
