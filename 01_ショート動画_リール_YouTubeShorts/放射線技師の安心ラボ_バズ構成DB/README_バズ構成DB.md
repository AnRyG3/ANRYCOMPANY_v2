# 放射線技師の安心ラボ バズ構成データベース

## 目的

「バズった理由」を再現可能なデータとして蓄積し、放射線系ショート動画を継続的に伸ばせる制作工場を構築する。

再生数、保存率、フォロワー増加、コメント、シリーズ化を総合的に分析し、「一般人が知らない」を武器にした動画構成を再利用可能にする。

## 対象テーマ

- 放射線
- レントゲン
- CT
- MRI
- 被ばく
- 医療雑学
- 整形撮影
- 医療系あるある
- 安心解説

## フォルダ構成

```text
放射線技師の安心ラボ_バズ構成DB/
├─ 01_バズDB/
├─ 02_フックDB/
├─ 03_保存型/
├─ 04_共感型/
├─ 05_驚き型/
├─ 06_医療雑学型/
├─ 07_安心解説型/
├─ 08_競合分析/
├─ 09_失敗分析/
└─ 10_再利用テンプレ/
```

## 運用フロー

1. 投稿テーマを決める
2. バズ動画分類から型を選ぶ
3. フックDBから3秒フックを選ぶ
4. バズ構成テンプレに当てはめる
5. サムネとタイトルを決める
6. 品質評価で90点以上まで改善する
7. 投稿後、結果をバズDBに記録する
8. 成功した動画はシリーズ化・派生化する
