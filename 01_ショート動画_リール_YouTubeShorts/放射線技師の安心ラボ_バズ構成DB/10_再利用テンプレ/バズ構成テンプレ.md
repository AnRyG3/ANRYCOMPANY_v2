# バズ構成テンプレ

## 基本構成

```text
0〜3秒:
強フック

3〜10秒:
結論

10〜20秒:
理由

20〜30秒:
比較・具体例

30〜40秒:
安心感

最後:
保存・コメント誘導
```

## 台本テンプレ

```text
{強フック}

結論、{結論}。

なぜなら、{理由}だからです。

たとえば、{比較・具体例}。

放射線技師の視点では、{技師視点}を見ています。

不安な時は、検査前に相談して大丈夫です。

あとで見返せるよう保存してね。
{コメント誘導}
```

## 型別の使い分け

| 型 | 冒頭 | 中盤 | 終盤 |
|---|---|---|---|
| 驚き型 | 実は | 意外な理由 | 知って安心 |
| 誤解破壊型 | それ誤解です | 正しい知識 | 保存 |
| 比較型 | AとBの違い | 比較表現 | 家族に共有 |
| 不安→安心型 | 不安代弁 | 結論 | 相談してOK |
| 技師視点型 | 現場では | 技師の見方 | フォロー |
| 医療あるある型 | あるある | 共感 | コメント |
| 初心者教育型 | 初心者向け | かんたん説明 | 保存 |
| 保存特化型 | 保存して | 3ポイント | 保存CTA |
