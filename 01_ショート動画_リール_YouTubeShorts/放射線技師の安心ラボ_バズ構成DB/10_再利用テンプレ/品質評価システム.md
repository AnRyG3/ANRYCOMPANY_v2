# バズ構成 品質評価システム

## 目的

投稿前の構成が、バズ候補として十分かを採点し、120点に近づくまで改善する。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| フック力 | /10 | 3秒以内に止まるか |
| 保存率期待 | /10 | 後で見返したくなるか |
| 完視聴期待 | /10 | 最後まで見たくなるか |
| コメント期待 | /10 | 反応したくなるか |
| 安心感 | /10 | 医療不安を下げられるか |
| 驚き | /10 | 一般人が知らない要素があるか |
| 一般人理解 | /10 | 難しい言葉が少ないか |
| 専門性 | /10 | 放射線技師視点があるか |
| シリーズ化 | /10 | 続編にできるか |
| 再利用性 | /10 | 型として使い回せるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 改善 | フック・結論・保存価値を見直す |
| 90点以上 | 投稿推奨 | 制作へ進む |
| 100点 | 強い | 優先投稿 |
| 120点 | バズ候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナスが強い場合、最大20点を加点して120点満点として扱う。

| ボーナス | 最大点 |
|---|---:|
| 最初の1秒が強い | +5 |
| 一般人が知らない要素が強い | +5 |
| 保存・コメント両方を狙える | +5 |
| シリーズ展開しやすい | +5 |
