# バズサムネDB

## 強い要素

| 要素 | 使い方 | 向いている型 |
|---|---|---|
| 赤枠 | 注目ポイントを囲む | 誤解・比較 |
| 黄色文字 | 重要語を強調 | 驚き型 |
| 矢印 | 見る場所を示す | 医療雑学・整形撮影 |
| 顔アップ | 感情を伝える | 不安→安心 |
| 不安ワード | 危険？怖い？ | 不安解説 |
| 比較構図 | CT vs MRI | 比較型 |
| 実は | 意外性 | 驚き型 |

## サムネ文言

- 危険？
- 実は
- 9割知らない
- それ誤解
- CTとMRI
- 放射線の誤解
- 技師はここを見る
- 検査前に見て

## 注意

- 不安ワードは使いすぎない
- 医療不安を煽るだけで終わらせない
- 最後は安心に着地する内容とセットにする
