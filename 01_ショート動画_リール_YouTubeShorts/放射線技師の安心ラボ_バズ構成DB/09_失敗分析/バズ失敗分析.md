# バズ失敗分析

## 失敗原因

| 原因 | 状態 | 改善 |
|---|---|---|
| 専門用語多すぎ | 一般人が離脱 | かんたんな言葉に言い換える |
| フック弱い | 冒頭で止まらない | 誤解・数字・比較を使う |
| 結論遅い | 途中で離脱 | 10秒以内に結論 |
| 情報薄い | 保存されない | 技師視点かチェックリストを入れる |
| 不安だけ煽る | 信頼低下 | 安心感で終わる |
| オチ弱い | 最後まで見ない | 保存・コメント誘導を入れる |

## 失敗分析テンプレ

```markdown
## 投稿テーマ


## 失敗原因


## 改善する場所

- フック:
- 結論:
- 理由:
- 比較:
- 安心感:
- CTA:

## 再投稿するなら


```
