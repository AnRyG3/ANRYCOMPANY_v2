# 保存率が低い原因分析

## 主な原因

| 原因 | 状態 | 改善 |
|---|---|---|
| 情報が浅い | 見返す価値がない | チェックリスト化する |
| テンポ悪化 | 説明が長い | 1文を短くする |
| 専門用語過多 | 一般人が理解できない | 言い換えを入れる |
| フック弱い | 冒頭で止まらない | 誤解・数字・比較を入れる |
| オチ不足 | 最後まで見る理由がない | 安心感と保存CTAを入れる |
| 価値不足 | 保存する必要がない | 検査前・家族向け・職場向けに寄せる |
| 見返す必要性がない | 雑学で終わる | 実用ポイントを3つ入れる |

## 改善テンプレ

```markdown
## 低保存率の原因


## 修正する場所

- フック:
- 構成:
- テロップ:
- サムネ:
- 保存誘導:

## 次回の改善案


```
