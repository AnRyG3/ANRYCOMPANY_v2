# 保存されやすい動画分析

## 冒頭フック

保存されやすいフックは、ただ驚かせるだけでなく「あとで使えそう」と思わせる。

## 強いフック例

| 型 | 例 | 保存理由 |
|---|---|---|
| 誤解訂正 | 実はこれ誤解です | 知識として残したい |
| 大多数 | 9割が知らない | 自分も知らないかもと思う |
| 不安代弁 | 放射線って危険と思ってませんか？ | 不安解消で保存される |
| 検査前 | 検査前にこれだけ見て | 後で見返したい |
| 比較 | CTとMRI、違いはここ | 家族に説明しやすい |
| 技師視点 | 技師はここを見ています | 現場感がある |

## 保存理由

- 後で見返したい
- 家族に見せたい
- 知識として残したい
- 不安解消になる
- 医療系あるあるとして共感できる
- 検査前の確認に使える
- 職場で説明に使える

## ジャンル別保存されやすさ

| ジャンル | 保存理由 |
|---|---|
| 被ばく | 不安解消、家族に説明したい |
| CT | 検査前に見返したい |
| MRI | CTとの違いを残したい |
| 整形撮影 | 技師が現場で見返したい |
| 雑学 | 知識として残したい |
| 誤解解説 | 間違った理解を直したい |
