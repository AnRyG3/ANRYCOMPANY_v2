# 放射線技師の安心ラボ 保存率分析システム

## 目的

「なぜ保存されたのか」を分析し、再現可能な保存特化リール制作工場を構築する。

再生数だけで終わらせず、教育系リールとして「後で見返したい」「家族に見せたい」「検査前に確認したい」を作る。

## 対象テーマ

- レントゲン
- CT
- MRI
- 被ばく
- 放射線の誤解
- 整形撮影
- 医療雑学
- 診療放射線技師視点

## フォルダ構成

```text
放射線技師の安心ラボ_保存率分析システム/
├─ 01_投稿分析/
├─ 02_保存率DB/
├─ 03_フック分析/
├─ 04_尺分析/
├─ 05_サムネ分析/
├─ 06_コメント分析/
├─ 07_競合比較/
├─ 08_改善案/
├─ 09_成功パターン/
└─ 10_失敗パターン/
```

## 運用フロー

1. 投稿後24時間、3日後、7日後に数値を記録する
2. 保存率を主指標として確認する
3. フック、構成、サムネ、尺、コメントを分析する
4. 保存された理由を言語化する
5. 成功パターンDBに追加する
6. 次回台本に保存される要素を1つ以上入れる

## 最重要ルール

- 保存率を主指標にする
- 保存理由を必ず書く
- 伸びた投稿は構成まで分解する
- 伸びなかった投稿は原因を1つに絞る
- 次回改善に必ずつなげる
