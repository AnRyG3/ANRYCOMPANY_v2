# 投稿改善システム

## 目的

投稿後に、改善点・保存率予測・離脱ポイント・再編集案・次回改善を自動生成する。

## 再編集判断

| 状態 | 対応 |
|---|---|
| 保存率高・再生数低 | サムネとフックを変えて再投稿候補 |
| 保存率低・再生数高 | 内容は広いが保存価値不足。チェックリスト化 |
| 完視聴率低 | 尺短縮、構成改善 |
| コメント多い | 次回テーマ化 |
| フォロー率高 | シリーズ化 |
