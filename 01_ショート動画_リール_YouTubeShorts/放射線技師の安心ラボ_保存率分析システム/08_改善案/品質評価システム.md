# 保存率品質評価システム

## 目的

保存される投稿になっているかを自己採点し、120点に近づくまで改善する。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| 保存価値 | /10 | 後で見返したいか |
| 安心感 | /10 | 不安が下がるか |
| 見返し価値 | /10 | 検査前・家族説明に使えるか |
| 専門性 | /10 | 診療放射線技師視点があるか |
| 一般人理解 | /10 | 専門用語が少ないか |
| コメント誘導 | /10 | 質問や経験談が出やすいか |
| サムネ力 | /10 | 保存したくなる入口か |
| テンポ | /10 | 最後まで見やすいか |
| 教育価値 | /10 | 知識として残るか |
| シリーズ化 | /10 | 次回に展開できるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 改善 | 保存理由と構成を作り直す |
| 90点以上 | 投稿推奨 | 制作へ進む |
| 100点 | 強い | 優先投稿候補 |
| 120点 | 保存バズ候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナスが強い場合、最大20点を加点して120点満点として扱う。

| ボーナス | 最大点 |
|---|---:|
| 検査前に見返す価値が高い | +5 |
| 家族に見せたくなる | +5 |
| 技師視点が強い | +5 |
| シリーズ化しやすい | +5 |
