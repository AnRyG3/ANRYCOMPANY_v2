# Instagramリール 勝ちパターン分析システム

## 目的

放射線技師アカウント「放射線技師の安心ラボ」で、再生数・保存率・フォロー率が高くなる傾向を分析し、今後の動画制作に反映する。

対象は、医療系リール、教育系ショート動画、雑学系リール、放射線関連、共感系動画。

## 分析の基本方針

単に再生数だけを見ない。

このアカウントでは、以下の3つを分けて評価する。

| 指標 | 意味 | 重視する動画 |
|---|---|---|
| 再生数 | 冒頭の強さ、テーマの広さ | 雑学、誤解解消、あるある |
| 保存率 | 実用性、後で見返す価値 | 被ばく、CT、MRI、検査前チェック |
| フォロー率 | 信頼感、専門性、継続視聴したい理由 | 技師視点、安心解説、シリーズ企画 |

## 使い方

1. 投稿後24時間、3日後、7日後に数値を記録する
2. フック、構成、感情、専門性、保存理由を分析する
3. 伸びた投稿を「勝ちパターン」として保存する
4. 次回台本に、勝ちパターンを1つ以上反映する

## 1. フック分析

## 伸びる言い回し

放射線技師アカウントでは、「怖い」だけで引っ張るよりも、不安を代弁してすぐ安心に向かう言い回しが強い。

| 型 | 例 | 使いどころ |
|---|---|---|
| 誤解訂正型 | レントゲン、実はここを誤解されがちです | 被ばく、CT、MRI |
| 不安代弁型 | CTが怖い人、これだけ知ってください | 一般向け |
| 意外性型 | 実は、レントゲンで全部は見えません | 雑学 |
| 現場視点型 | 技師は撮影中、ここを見ています | 技師あるある、教育 |
| 保存前提型 | 検査前にこれだけ保存して | チェックリスト系 |
| 比較型 | CTとMRI、いちばん違うのはここ | 初心者向け |

## 離脱されやすい言い回し

| NG表現 | 理由 | 改善例 |
|---|---|---|
| 今日はCTについて説明します | 普通すぎて止まらない | CTが怖い人、まずここだけ知って |
| 放射線被ばくとは | 教科書感が強い | 被ばくって聞くと怖いですよね |
| 詳しく解説します | 長そうに見える | 30秒でわかるように話します |
| 医学的には | 難しそうに見える | かんたんに言うと |
| 絶対安全です | 医療炎上リスク | 必要な検査なら過度に怖がりすぎなくて大丈夫 |

## “え？”と思わせる言葉

- 実は
- 意外と
- それ、誤解です
- 全部見えるわけではありません
- 技師はここを見ています
- 撮り直しの原因、実はこれです
- CTとMRI、同じだと思っていませんか？
- レントゲン前に外す理由があります
- 怖がるポイント、そこではないかもしれません

## 保存されやすい始まり方

- 検査前にこれだけ確認して
- 家族に説明するならこれ
- レントゲン前に外すものリスト
- CTが不安な人のための3つのポイント
- 妊娠中に検査が不安な人へ
- 子どもの検査で親が知っておきたいこと

## 2. 構成分析

## 最適秒数

| 目的 | 推奨秒数 | 理由 |
|---|---:|---|
| 高再生狙い | 15〜22秒 | 最後まで見られやすい |
| 高保存狙い | 25〜35秒 | 実用情報を入れやすい |
| 高フォロー狙い | 25〜35秒 | 専門性と安心感を出せる |
| 技師あるある | 12〜20秒 | 共感でテンポよく見せる |
| 撮影法解説 | 25〜35秒 | 手順とチェックポイントが必要 |

## 結論を入れる位置

結論は5〜8秒以内に入れる。

理想は以下。

```text
0〜2秒: フック
2〜5秒: 不安・誤解
5〜8秒: 結論
8〜25秒: 理由と技師視点
25〜35秒: 安心感と保存CTA
```

## テンポ

- 1文は短くする
- 1文1メッセージにする
- 3秒以内に画面変化を入れる
- 難しい話は図解に逃がす
- 口頭で言う内容とテロップを完全一致させすぎない

## カット頻度

| 秒数 | 推奨 |
|---|---|
| 0〜3秒 | 1秒ごとに変化 |
| 3〜10秒 | 2秒ごとに変化 |
| 10〜25秒 | 2〜3秒ごとに変化 |
| ラスト | CTAを1.5〜2秒見せる |

## テロップ量

- 1画面最大2行
- 1行12〜16文字目安
- 重要語だけ青で強調
- 画面の3分の1以上を文字で埋めない
- 医療系は読みにくさが信頼低下につながる

## 3. 感情分析

## 最適な感情の順番

一般向けの放射線リールは、以下の順番が強い。

```text
不安の代弁
↓
驚き
↓
理解
↓
安心
↓
信頼
↓
保存
```

## 感情設計テンプレ

| 秒数 | 感情 | 例 |
|---:|---|---|
| 0〜3秒 | 不安・驚き | CTって聞くと怖いですよね |
| 3〜8秒 | 誤解の整理 | でも、怖がるポイントはそこだけではありません |
| 8〜18秒 | 理解 | 必要な検査は、情報を得るメリットがあります |
| 18〜28秒 | 安心 | 不安な時は相談して大丈夫です |
| 28〜35秒 | 保存・信頼 | 検査前に見返せるよう保存してね |

## 避けたい感情順

```text
恐怖
↓
専門用語
↓
断定
↓
放置
```

これは不安だけ残して離脱や炎上につながりやすい。

## 4. 放射線技師特化分析

## 一般人が知らない知識

- レントゲンは何でも写るわけではない
- 撮影で息を止めるのはブレを防ぐため
- 金属や湿布は画像に写り込むことがある
- CTとMRIは得意なものが違う
- 検査ごとに必要最小限の条件を考えている
- 同じ部位でも目的によって撮り方が変わる
- 立位荷重でないと分からない情報がある
- 撮り直しは患者さんの負担にもつながる

## 技師視点

- 撮影中に見ているのは「きれいな写真」だけではない
- 診断に必要な範囲が入っているかを見ている
- 体位が崩れると画像の意味が変わる
- 患者さんの痛みや動ける範囲も見ている
- 再現性は経過観察で重要
- 不安な患者さんには声かけが大事

## 医師との違い

| 医師 | 放射線技師 |
|---|---|
| 診断・治療方針を決める | 診断に必要な画像を撮る |
| 病気の判断をする | 目的に合った撮影を組み立てる |
| 検査の必要性を判断する | 安全で見やすい画像を作る |

表現注意:
医師との比較は対立ではなく、役割の違いとして説明する。

## “現場感”の強さ

強い現場感:

- 撮影中、技師はここを見ています
- 実は撮り直しになりやすいのはこの動き
- 患者さんにこれをしてもらえると助かります
- 膝の撮影で足先の向きが大事な理由

弱い現場感:

- CTとは何か
- レントゲンとは何か
- 被ばくについて説明します

## 5. 保存率分析

## 保存される理由

| 理由 | 強いテーマ |
|---|---|
| 検査前に見返したい | CT、MRI、レントゲン前チェック |
| 家族に説明したい | 被ばく、子ども、妊娠中 |
| 職場で使える | 撮影法、声かけ、チェックポイント |
| 一覧性がある | 外すものリスト、比較表 |
| 不安な時に安心したい | 被ばく、検査の安全性 |

## 高保存率の形

```text
不安な人へ
↓
結論
↓
確認ポイント3つ
↓
注意点
↓
不安なら相談してOK
↓
保存CTA
```

## 保存率が下がる形

- 結論だけで終わる
- 雑学だけで実用性がない
- 情報が多すぎて覚えられない
- 専門用語が多い
- 保存CTAがない

## 6. コメント誘導分析

## コメントしやすい問いかけ

| 型 | 例 |
|---|---|
| 経験談 | レントゲンで不安だったことはありますか？ |
| 誤解確認 | CTとMRI、同じだと思っていましたか？ |
| 選択式 | 怖いのはCT？MRI？ |
| 医療職向け | 技師さん、この説明どうしていますか？ |
| 地域差 | あなたの病院ではこの撮影、どうしていますか？ |
| 次回希望 | 次に知りたい検査をコメントで教えてください |

## 地域差・施設差の扱い

撮影法や施設運用は地域差・施設差がある。

コメントを増やしやすい言い方:

```text
施設によって少し違うこともあります。
みなさんの職場ではどうしていますか？
```

炎上しやすい言い方:

```text
この撮り方が正解です。
これ以外は間違いです。
```

## 7. バズ危険分析

## 医療炎上リスク

| 危険 | 例 | 回避表現 |
|---|---|---|
| 強すぎる断定 | 絶対安全です | 過度に怖がりすぎなくて大丈夫です |
| 不安煽り | CTは危険です | CTには役割があり、必要性を見て行われます |
| 個別診断 | これは病気です | 気になる場合は医師に相談してください |
| 医師否定 | 医師より技師が分かる | 技師は画像を作る専門職です |
| 施設否定 | その撮り方は間違い | 施設基準により異なる場合があります |

## 投稿前リスクチェック

- [ ] 絶対、必ず、危険、安全と言い切っていない
- [ ] 医療不安を煽りすぎていない
- [ ] 個別診断に見えない
- [ ] 医師や他職種を否定していない
- [ ] 施設差のある内容を断定していない
- [ ] 最後に相談先や安心感がある
- [ ] コメントで誤解されそうな表現がない


## “放射線技師だから強い”ポイント一覧

- 検査室の中で実際に患者さんを見ている
- 撮影条件や体位の意味を説明できる
- レントゲン、CT、MRIの違いを現場目線で話せる
- 患者さんが不安になりやすい場面を知っている
- 医師とは違う「画像を作る専門職」としての視点がある
- 撮り直しが起きる原因を具体的に説明できる
- 整形撮影法の実践的なコツを出せる
- 医療職向けと一般向けの両方に展開できる
- 検査前の不安をやさしく下げる立ち位置を取れる
