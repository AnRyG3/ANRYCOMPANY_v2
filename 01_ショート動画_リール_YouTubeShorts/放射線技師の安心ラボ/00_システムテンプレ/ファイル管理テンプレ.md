# ファイル管理テンプレ

## 1投稿ごとの構成

```text
Reels/
└─ YYYY-MM-DD_テーマ名/
   ├─ script.txt
   ├─ prompt.txt
   ├─ voice.wav
   ├─ thumb.png
   ├─ caption.txt
   ├─ final.mp4
   └─ analysis.md
```

## 命名ルール

```text
YYYY-MM-DD_カテゴリ_テーマ名
```

## 例

```text
2026-05-11_CT_CTは危険なのか
2026-05-13_レントゲン_骨が白く写る理由
2026-05-15_撮影法_ローゼンバーグ撮影
```

## 各ファイルの役割

| ファイル | 内容 |
|---|---|
| script.txt | 台本、テロップ、ナレーション |
| prompt.txt | Gemini画像生成プロンプト |
| voice.wav | VOICEVOX音声 |
| thumb.png | サムネイル |
| caption.txt | 投稿文、ハッシュタグ、固定CTA |
| final.mp4 | 投稿する完成動画 |
| analysis.md | 投稿後分析 |
