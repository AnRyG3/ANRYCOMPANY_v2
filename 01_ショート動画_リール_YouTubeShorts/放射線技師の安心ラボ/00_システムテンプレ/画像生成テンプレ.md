# 画像生成テンプレ

## Gemini用 基本プロンプト

```text
Instagramリール動画用の医療系ビジュアルを作成してください。

テーマ:
{テーマ}

条件:
- 日本人向け
- 医療系アカウント向け
- Instagram映えする
- 9:16比率
- 白背景ベース
- 青・水色系
- 清潔感がある
- シンプルで見やすい
- 怖く見せない
- 不安を煽らない
- 文字は入れない
- Canvaでテロップを入れやすい余白を残す
- 背景や図解自体が説明している場合は、その意味が隠れない構図にする
- 1画面1メッセージで、視線が散る装飾やラベルを増やしすぎない

内容:
- 放射線技師または医療スタッフがやさしく説明している雰囲気
- 一般の人にも伝わるやわらかい医療イラスト
- 必要に応じてレントゲン、CT、MRI、病院、検査室の要素を入れる
```

## CT・被ばく系

```text
CT検査と放射線被ばくについて、一般の人が安心して理解できるInstagramリール用イラストを作成してください。

デザイン:
- 9:16
- 白背景
- 青・水色系
- 清潔感のある医療デザイン
- 怖い表現を避ける
- 放射線を危険そうに描かない
- 医療スタッフがやさしく説明している雰囲気
- 文字なし
- 図解の重要部分を後入れテロップで隠さない余白設計にする
```

## 整形撮影法系

```text
整形外科のX線撮影法を説明するInstagramリール用イラストを作成してください。

撮影法:
{撮影法名}

条件:
- 9:16
- 白背景
- 青・水色系
- シンプルな図解風
- 患者さんの姿勢が分かる
- 関節の角度や体位が伝わる
- 医療従事者にも一般の人にも分かりやすい
- 怖い雰囲気にしない
- 文字なし
- 体位や角度など、画像で伝えるべき要素をテロップ予定位置で隠さない
```

## サムネ用

```text
Instagramリールのサムネイル背景として使う医療系イラストを作成してください。

テーマ:
{テーマ}

条件:
- 9:16
- 中央に主役の医療イラスト
- 上部に大きな文字を入れる余白
- 下部に補足文字を入れる余白
- 白背景
- 青・水色系
- 清潔感
- シンプル
- 一般の人が怖がらない雰囲気
- 画像内に文字は入れない
```
