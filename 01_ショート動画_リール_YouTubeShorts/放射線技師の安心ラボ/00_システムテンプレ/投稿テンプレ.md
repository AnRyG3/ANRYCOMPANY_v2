# 投稿テンプレ

## 投稿文テンプレ

```text
【{テーマ}】

{フック}

結論、{結論}

放射線検査は「怖いもの」と思われがちですが、
正しく知ることで不安はかなり減らせます。

不安があるときは、検査前に医師やスタッフへ相談して大丈夫です。

あとで見返せるように保存してね。

気になる検査や不安なことがあれば、コメントで教えてください。
```

## 固定CTA

```text
放射線検査の不安を減らす投稿をしています。
保存して、必要なときに見返してください。
```

## コメント誘導例

- レントゲンで不安だったことはありますか？
- CTとMRI、どちらが怖いイメージですか？
- 検査前に聞きたいことがあればコメントしてください。
- 医療職の方は、患者さんにどう説明していますか？
- この撮影、経験ありますか？
