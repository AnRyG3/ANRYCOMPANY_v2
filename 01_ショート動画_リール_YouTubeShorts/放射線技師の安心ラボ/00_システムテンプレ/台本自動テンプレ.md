# 台本自動テンプレ

## 入力

```text
テーマ:
ターゲット:
カテゴリ:
伝えたい結論:
入れたい技師視点:
動画尺: 15〜35秒
```

## 出力フォーマット

```markdown
# リール台本

## フック

0〜3秒:

## 本文

3〜8秒:

8〜15秒:

15〜25秒:

## オチ

25〜30秒:

## 安心感

30〜33秒:

## 保存CTA

33〜35秒:

## コメント誘導


## 画面テロップ


## ナレーション


## サムネ文言


## 投稿文の一言目


```

## 台本生成プロンプト

```text
以下のテーマでInstagramリール台本を作成してください。

アカウント:
放射線技師の安心ラボ

テーマ:
{テーマ}

条件:
- 15〜35秒
- 冒頭3秒で興味を引く
- 最初の1秒に「え？」と思う要素を入れる
- 結論を早めに出す
- 難しい言葉を使わない
- 一般の人の誤解を1つ入れる
- 放射線技師しか知らない視点を1つ入れる
- 最後は安心感で終わる
- 保存CTAを入れる
- コメントしたくなる問いかけを入れる
- 医療不安を過剰に煽らない

出力:
- フック
- 本文
- オチ
- 保存CTA
- コメント誘導
- テロップ
- ナレーション
```
