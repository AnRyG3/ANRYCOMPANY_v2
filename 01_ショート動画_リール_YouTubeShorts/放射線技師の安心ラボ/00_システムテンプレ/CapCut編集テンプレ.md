# CapCut編集テンプレ

## 基本設定

| 項目 | 設定 |
|---|---|
| 比率 | 9:16 |
| 解像度 | 1080x1920 |
| 尺 | 15〜35秒 |
| 形式 | MP4 |
| 音声 | VOICEVOX中心 |

## カットテンポ

- 冒頭3秒は1秒ごとに画面変化
- 1カットは1〜3秒
- 説明部分は図解やズームで変化をつける
- 重要語は画面中央に大きく出す
- 最後は少し落ち着いたテンポで安心感を出す

## テロップルール

- 1画面最大2行
- 1行12〜16文字程度
- 文字は大きく太く
- 白背景に濃いグレー文字
- 重要語だけ青・水色
- 注意はやわらかい赤かオレンジ
- 専門用語はすぐ言い換える
- 小さい文字で長文を詰め込まない
- 1画面1メッセージにする
- 背景や図解が説明している時は、テロップで大事な情報を隠さない
- カテゴリラベルは必要な時だけ使い、視線を増やしすぎない
- 画像は説明、テロップは短い補助として使う

## BGMルール

- ナレーションを邪魔しない
- 音量は小さめ
- 医療不安を煽るBGMは避ける
- 清潔感、安心感、やさしさを優先
- 技師あるあるは少し軽めでもよい
- CT・被ばく系は落ち着いたBGM

## SEルール

- 冒頭フックに軽い効果音を1つ
- 強調箇所に短いSE
- 多用しない
- 不安を煽る警告音は使わない
- 医療情報の信頼感を壊さない

## 編集チェック

- [ ] 冒頭1秒で内容が分かる
- [ ] 3秒以内に結論の方向性が見える
- [ ] 文字が大きく読める
- [ ] 画面がごちゃついていない
- [ ] テロップが背景や図解の重要部分を隠していない
- [ ] 不要なカテゴリラベルで視線が散っていない
- [ ] ナレーションが聞き取りやすい
- [ ] 最後に安心感がある
- [ ] 保存CTAが入っている
