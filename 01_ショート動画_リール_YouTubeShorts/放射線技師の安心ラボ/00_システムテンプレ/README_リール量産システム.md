# Instagramリール量産システム

## 目的

放射線技師アカウント「放射線技師の安心ラボ」のリール動画を、毎回ゼロから考えずに週3〜5本で継続投稿できる状態にする。

## 運用の流れ

1. `01_テーマ管理/テーマ管理シート.md` からテーマを選ぶ
2. `00_システムテンプレ/台本自動テンプレ.md` にテーマを入れて初稿を作る
3. `00_システムテンプレ/台本品質評価システム.md` で採点する
4. 120点に近づくまで改善する
5. `00_システムテンプレ/画像生成テンプレ.md` でGemini用画像を作る
6. `00_システムテンプレ/CapCut編集テンプレ.md` に沿って編集する
7. `00_システムテンプレ/投稿テンプレ.md` で投稿文とハッシュタグを作る
8. 投稿後に `02_分析/バズ分析テンプレ.md` へ結果を記録する

## 1投稿ごとの保存場所

```text
Reels/
└─ YYYY-MM-DD_テーマ名/
   ├─ script.txt
   ├─ prompt.txt
   ├─ voice.wav
   ├─ thumb.png
   ├─ caption.txt
   ├─ final.mp4
   └─ analysis.md
```

## 最重要ルール

- 3秒で興味を引く
- 難しい言葉を使わない
- 結論を早めに出す
- 最後は安心感で終わる
- 保存したくなる構成にする
- 医療不安を煽りすぎない
- 放射線技師視点を必ず入れる
- コメントしたくなる問いかけを入れる

## 量産時の基本配分

| 曜日 | 投稿ジャンル |
|---|---|
| 月曜 | 一般向けの不安解消 |
| 火曜 | レントゲン・CT豆知識 |
| 水曜 | 技師あるある |
| 木曜 | 撮影法・医療従事者向け |
| 金曜 | 医療雑学・保存系まとめ |

週3本の場合は、月曜・水曜・金曜を基本にする。
