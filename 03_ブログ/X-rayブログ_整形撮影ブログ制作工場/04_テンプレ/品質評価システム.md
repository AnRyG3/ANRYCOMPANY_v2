# 記事品質評価システム

## 目的

整形領域X線撮影ブログを、検索流入と保存価値の高い記事へ改善する。

AIが自己採点し、改善案を提示し、120点に近づくまで仮想改善を繰り返す。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| わかりやすさ | /10 | 若手技師が読んで理解できるか |
| 実用性 | /10 | 撮影時に見返して使えるか |
| SEO | /10 | 検索キーワードが自然に入っているか |
| 初心者理解度 | /10 | 新人でも迷わない説明か |
| 保存価値 | /10 | ブックマークしたくなるか |
| 画像必要性 | /10 | どこに画像が必要か明確か |
| 専門性 | /10 | 技師向けの深さがあるか |
| 現場感 | /10 | 実際の撮影で役立つ視点があるか |
| 内部リンク性 | /10 | 関連記事へつなげやすいか |
| 再訪価値 | /10 | 後から何度も見返したくなるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 修正 | 構成・説明・チェックポイントを見直す |
| 90点以上 | 公開可能 | 公開してよい |
| 100点 | 強い記事 | 優先公開候補 |
| 120点 | 検索上位候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナス要素が強い場合、最大20点を加点して120点満点として扱う。

| ボーナス項目 | 最大点 |
|---|---:|
| 撮影イメージが非常に分かりやすい | +5 |
| チェックポイントが保存向き | +5 |
| 医師が見ているポイントが具体的 | +5 |
| 関連記事への展開性が高い | +5 |

## 改善フロー

1. 初稿を作成
2. 10項目で自己採点
3. 80点未満なら修正
4. 90点以上なら公開可能
5. ボーナス加点を狙って改善
6. 120点に近づくまで、撮影イメージ・チェックポイント・SEO・内部リンクを強化
