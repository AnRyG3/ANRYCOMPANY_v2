# 整形領域X線撮影ブログ 完全テンプレートシステム

## 目的

若手診療放射線技師向けに、整形領域X線撮影ブログを量産するための制作システム。

部位や撮影法を差し替えるだけで、検索流入を狙える、撮影時に見返せる、保存価値の高い記事を作る。

## 対象ジャンル

- 股関節
- 膝関節
- 足関節
- 足部
- 肩関節
- 手関節
- 肘関節
- 脊椎
- 特殊撮影
- マニアック撮影法

## フォルダ構成

```text
X-rayブログ_整形撮影ブログ制作工場/
├─ 01_部位別/
├─ 02_画像/
├─ 03_撮影イメージ/
├─ 04_テンプレ/
├─ 05_SEO/
├─ 06_サムネ/
├─ 07_公開済/
└─ 08_リライト/
```

## 制作フロー

1. `01_部位別` に部位別記事を作る
2. `04_テンプレ/記事マスターテンプレート.md` をコピーする
3. 撮影名、目的、体位、入射条件を差し替える
4. `03_撮影イメージ` に掲載予定画像の指示を作る
5. `05_SEO/SEOテンプレ.md` で検索キーワードを整理する
6. `06_サムネ/サムネテンプレ.md` でサムネ案を作る
7. `04_テンプレ/品質評価システム.md` で自己採点する
8. 90点以上なら公開、120点候補なら優先公開する
9. 公開後は `07_公開済` に保管する
10. 検索順位やクリック率を見て `08_リライト` で改善する

## 記事量産のルール

- 撮影イメージを最優先に置く
- 若手技師が現場で見返せる文章にする
- 画像が必要な場所を明記する
- 医師が見ているポイントを必ず入れる
- 失敗例とチェックポイントを厚くする
- SEOタイトルとメタディスクリプションを毎回作る
- 関連記事リンクで部位ごとの回遊を作る
- 施設差がある内容は断定しすぎない
