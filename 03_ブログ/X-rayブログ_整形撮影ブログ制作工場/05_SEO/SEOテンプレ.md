# SEOテンプレ

## SEOタイトル

```text
{撮影名}とは？撮影方法・入射角・チェックポイントを技師向けに解説
```

### タイトル候補

1. {撮影名}とは？撮影目的・ポジショニング・チェックポイント
2. {撮影名}の撮り方｜入射角・中心線・整位のコツ
3. {部位}X線撮影のポイント｜{撮影名}を若手技師向けに解説
4. {撮影名}で失敗しやすいポイントと再撮影を減らすコツ
5. {撮影名}のチェックポイント｜医師が見ている画像所見も解説

## メタディスクリプション

```text
{撮影名}の目的、ポジショニング、X線入射条件、チェックポイント、よくある失敗を若手診療放射線技師向けに解説。撮影時に見返せる実践的なコツもまとめています。
```

## 見出し構成

```text
H1: {撮影名}とは？撮影目的・ポジショニング・チェックポイントを解説
H2: 撮影イメージ
H2: この撮影の目的
H2: ポジショニング

H2: 医師が見ているポイント

H2: まとめ
H2: 関連記事
```

## SEOチェックリスト

- [ ] タイトルに撮影名が入っている
- [ ] タイトルに「撮り方」「チェックポイント」「ポジショニング」のどれかが入っている
- [ ] 冒頭100文字以内に撮影名が入っている
- [ ] H2に撮影名または部位名が自然に入っている
- [ ] 画像altに撮影名と部位名が入っている
- [ ] 関連記事リンクが3つ以上ある
- [ ] 若手技師が検索しそうな言葉を使っている
