# サムネテンプレ

## 目的

検索結果、SNS、ブログ内関連記事でクリックされやすい整形X線撮影ブログ用サムネを作る。

## 基本構成

```text
上部: 撮影名
中央: 撮影イメージまたはX線画像
右側: 赤枠・矢印・注目ポイント
下部: 初心者向けの一言
```

## クリックされやすい文言

- 失敗しやすい
- ここを見る
- 整位のコツ
- 入射角のポイント
- 新人向け
- 再撮影を減らす
- チェックポイント
- 医師はここを見る
- 角度ミス注意
- 回旋ズレ注意

## デザインルール

- 赤枠で注目部位を囲む
- 矢印で中心線や評価部位を示す
- 文字は大きく
- 背景は白または薄いグレー
- 医療系らしい清潔感を出す
- 初心者向け感を出す
- ごちゃごちゃさせない
- 1サムネ1メッセージにする

## サムネ案テンプレ

```markdown
## 撮影名


## メインコピー


## サブコピー


## 赤枠で囲む場所


## 矢印で示す場所


## 使用画像


## 注意点

- [ ] 文字が小さすぎない
- [ ] 赤枠が多すぎない
- [ ] 何の記事か一瞬で分かる
- [ ] 初心者がクリックしたくなる
```
