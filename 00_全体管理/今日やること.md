## 開始合図テンプレ

Codexを立ち上げただけでは制作は始めない。
制作を始める時は、次のどちらかを入力する。

```text
今日はリール制作です。いつもの運用でお願いします。
```

```text
ANRYCAMPANY、今日の制作を始めてください。
```

## いつもの運用でCodexが出すもの

```text
今日のメイン成果：
リール台本案を提出

Codexからの追加提案：
LINEスタンプ候補 10個
ブログ記事候補 5個
リールからの横展開候補 3個

あなたが判断すること：
リール台本のOK / 修正
スタンプ案の採用 / 修正 / 保留
ブログ案の採用 / 修正 / 保留
横展開候補の採用 / 修正 / 保留
```

## リール制作の注意

- リールはいきなり制作に入らない
- まず台本を提出する
- 医療従事者目線と患者目線で食い違いがないか確認する
- ユーザーがOKを出した後に、画像、音声、動画、投稿文の制作へ進む
