# ANRYCAMPANY

ANRYCAMPANYの制作データ本体です。作業場所は `F:\ANRYCAMPANY` です。

## まず見る場所

- `00_全体管理` : 方針、決定事項、今日やること
- `01_ショート動画_リール_YouTubeShorts` : 投稿企画、完成形、リール制作単位の資料
- `02_LINEスタンプ` : LINEスタンプ制作データ、提出用素材
- `03_ブログ` : ブログ用の原稿や素材
- `reel_assets` : 動画生成用の画像、音声、CapCut出力、共通素材
- `SNS-OS` : SNS運用、分析、投稿管理の仕組み
- `tools` : 生成・変換・整理に使うスクリプトや外部ツール

## 完成品を探すとき

- InstagramやYouTube Shorts用の完成動画: `01_ショート動画_リール_YouTubeShorts\インスタ完成形`
- CapCutから書き出した動画: `reel_assets\capcut_exports`
- LINEスタンプ提出用: `02_LINEスタンプ\LINEスタンプ`

## 制作素材を探すとき

- CTリール素材: `reel_assets\ct_*`
- 共通エンドカード: `reel_assets\common`
- 使わない素材: `reel_assets\_不採用_今後使わない`

## 生成スクリプト

CT動画などの生成スクリプトは `tools\ct_video_builders` にまとめました。
ルート直下には制作物の入口だけを残す方針です。
