# LINEスタンプ制作 作業テンプレート

## 目的

チワワキャラクター「あんりぃ」のLINEスタンプを量産・統一管理する。

キャラクターデザイン、色味、文字入れ、サイズ、ファイル名を統一し、シリーズ展開しやすい制作体制を作る。

## 基本仕様

| 種類 | ファイル名 | サイズ | 形式 | 備考 |
|---|---|---:|---|---|
| メイン画像 | main.png | 240x240px | PNG透過 | スタンプ一覧や販売ページ用 |
| トークルームタブ画像 | tab.png | 96x74px | PNG透過 | 小さく見ても判別できる絵柄 |
| スタンプ画像 | 01.png〜40.png | 370x320px | PNG透過 | 最大40個構成 |

### 共通ルール

- PNG透過で書き出す
- ZIP化してLINE Creators Marketへアップロードする
- 画像の余白は20%程度を目安にする
- 文字は読みやすさを優先する
- キャラクターと文字が端に寄りすぎないようにする
- 白フチ、淡い影、縁取りなどで視認性を確保する

## ファイル命名規則

### 提出用ファイル

LINE Creators Marketへ提出する最終ファイル名は、必ず以下に統一する。

```text
main.png
tab.png
01.png
02.png
03.png
...
40.png
```

### 作業中ファイル

作業中は内容が分かる名前を付け、最終書き出し時に提出用ファイル名へ変換する。

```text
anry_01_ありがとう.png
anry_02_おはよう.png
anry_03_おつかれさま.png
anry_04_了解.png
anry_05_ごめんね.png
```

### バージョン管理

修正が入った場合は、作業中ファイルのみバージョン番号を付ける。

```text
anry_01_ありがとう_v01.png
anry_01_ありがとう_v02.png
anry_01_ありがとう_final.png
```

最終提出フォルダには、バージョン番号を付けない。

## 作業手順テンプレ

## 1. セリフ設計

まず40個分のセリフを決める。

日常会話で使いやすいものを優先し、感情・挨拶・返事・リアクションをバランスよく入れる。

### セリフカテゴリ例

| カテゴリ | 例 |
|---|---|
| 挨拶 | おはよう、こんにちは、おやすみ |
| 返事 | OK、了解、はーい、ありがとう |
| 感情 | うれしい、かなしい、びっくり、しょんぼり |
| 応援 | がんばって、無理しないで、応援してる |
| 予定 | 今向かってる、あとで連絡する、少し待ってね |
| 女性向け | きゅん、かわいい、癒された、すき |

## 2. Geminiで画像生成

Geminiで9枚ずつ画像生成する。

40個制作する場合は、9枚単位で5回に分けて生成する。

```text
batch_01: 01〜09
batch_02: 10〜18
batch_03: 19〜27
batch_04: 28〜36
batch_05: 37〜40 + main + tab候補
```

生成後は、キャラクターデザインが崩れていない画像だけを採用する。

## 3. Canvaで文字入れ

採用した画像をCanvaに取り込み、セリフを入れる。

文字は大きく、スマホ画面で読めるサイズにする。装飾しすぎず、キャラクターのかわいさと読みやすさを両立する。

## 4. 背景透過PNG化

Canvaまたは画像編集ツールで背景を透過し、PNG形式で書き出す。

透過漏れ、白いフチの欠け、不要なゴミがないか確認する。

## 5. サイズ統一

提出用サイズに統一する。

```text
main.png: 240x240px
tab.png: 96x74px
01.png〜40.png: 370x320px
```

キャラクターや文字が切れていないか、余白が極端に少なくないか確認する。

## 6. ZIP化

`05_ZIP提出用` フォルダに提出用ファイルだけを入れ、ZIP化する。

ZIP内には余計なフォルダや作業中ファイルを入れない。

## 7. LINE Creators Marketへアップロード

LINE Creators Marketへログインし、スタンプ情報、販売エリア、価格、説明文、タグを設定してアップロードする。

アップロード後はプレビューで全スタンプを確認し、文字切れ・透過ミス・ファイル不足がないか確認する。

## Canva運用ルール

## デザインルール

- 背景は透過前提で作る
- キャラクターは中央寄せを基本にする
- 余白は20%程度を目安にする
- 文字は上部または下部に配置する
- キャラクターの顔に文字を重ねない
- 小さく表示されても読める太めのフォントを使う
- パステル系の色を基本にする
- 女性向けのかわいい印象を優先する
- 1セット内でフォントを統一する
- 文字色は白、ブラウン、ピンク、淡い赤などを中心にする
- 必要に応じて白フチや影を付ける

## Canva内の管理ルール

- 1セットごとにCanvaデザインを分ける
- ページ番号とスタンプ番号を一致させる
- 01ページ目は01.png、02ページ目は02.pngのように管理する
- main.pngとtab.pngは別ページで作る
- 書き出し前に全ページを一覧表示して、統一感を確認する

## 審査落ち回避ルール

## 画像面

- 透過PNGになっているか確認する
- 背景に白いゴミや消し残しがないか確認する
- 画像サイズが規定通りか確認する
- キャラクターや文字が切れていないか確認する
- 文字が小さすぎないか確認する
- 似たポーズや似た表情が多すぎないようにする
- スタンプとして使い道が分かる内容にする

## 内容面

- 暴力的、性的、差別的な表現を入れない
- 医療・診断・治療を断定する表現を入れない
- 著作権や商標に触れる言葉やデザインを使わない
- 既存キャラクターに似せすぎない
- 有名ブランド、作品名、芸能人名を使わない
- 誤字脱字を必ず確認する
- 使いにくい内輪ネタだけで構成しない

「あんりぃ」は単発スタンプではなく、継続して育てるキャラクターIPとして扱う。

- 名前: あんりぃ

- 印象: 小さくてかわいい、やさしい、癒し系
- 色味: パステル系

### 育て方

- シリーズ名に必ず「あんりぃ」を入れる
