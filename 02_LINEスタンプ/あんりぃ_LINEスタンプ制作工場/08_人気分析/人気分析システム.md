# 人気分析システム

## 目的

販売後の反応を分析し、次シリーズの売れやすさとIP育成に反映する。

## 分析項目

| 項目 | 見るポイント | 評価 |
|---|---|---|
| 使いやすさ | 日常で何度も送れるか | /10 |
| 送信頻度 | 毎日使える言葉が多いか | /10 |
| 女性ウケ | かわいさ、共感、色味 | /10 |
| 視認性 | 小さくても読めるか | /10 |
| 感情伝達力 | 表情と文字が一致しているか | /10 |
| 保存率 | 取っておきたいテーマか | /10 |
| 購入率 | タイトルとサムネで欲しくなるか | /10 |
| シリーズ化適性 | 次展開しやすいか | /10 |

## 人気が出やすい条件

- 7文字以内の短文が多い
- 返信に使える
- 感謝、挨拶、応援が入っている
- キャラ表情が大きく分かる
- 文字が読みやすい
- 色がやさしい
- 日常と感情のバランスが良い
- 推し活や恋愛など刺さるテーマがある

## 分析テンプレ

```markdown
# 人気分析

## シリーズ名


## 売れた理由


## 使われやすい文字


## 弱かった文字


## 女性ウケした要素


## 次シリーズに残すもの


## 次シリーズで変えるもの


## シリーズ化案


```
