# 品質評価システム

## 目的

あんりぃのLINEスタンプを販売前に自己採点し、人気候補・IP化候補まで引き上げる。

AIが自己採点し、改善案を提示し、120点に近づくまで仮想改善を繰り返す。

## 採点項目

各10点満点で採点する。

| 項目 | 点数 | 評価ポイント |
|---|---:|---|
| かわいさ | /10 | 一目でかわいいか |
| 統一感 | /10 | 同じキャラクターに見えるか |
| 使いやすさ | /10 | 日常会話で使えるか |
| 女性人気 | /10 | 10代後半〜40代女性に刺さるか |
| 感情伝達 | /10 | 表情と文字で感情が伝わるか |
| IP力 | /10 | あんりぃとして覚えられるか |
| 保存したくなるか | /10 | 取っておきたい魅力があるか |
| シリーズ化 | /10 | 次シリーズに展開できるか |
| 視認性 | /10 | 小さくても読めるか |
| 購入率予測 | /10 | 販売ページで欲しくなるか |
| 合計 | /100 |  |

## 判定基準

| 点数 | 判定 | 対応 |
|---:|---|---|
| 80点未満 | 修正 | キャラ統一、文字、使いやすさを見直す |
| 90点以上 | 販売可能 | 提出準備へ進む |
| 100点 | 人気候補 | 優先販売候補 |
| 120点 | バズ・IP化候補 | ボーナス込みの理想値 |

## 120点評価について

基本採点は100点満点。

以下のボーナスが強い場合、最大20点を加点して120点満点として扱う。

| ボーナス項目 | 最大点 |
|---|---:|
| 超使用頻度の短文が多い | +5 |
| あんりぃのIP認知が強い | +5 |
| 女性向けのかわいさが高い | +5 |
| シリーズ展開が明確 | +5 |
