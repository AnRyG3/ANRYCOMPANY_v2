# LINEスタンプ リジェクト対策一覧

## 画像チェック

| リスク | 内容 | 対策 |
|---|---|---|
| 透過不備 | 背景が残っている | PNG透過を確認する |
| 余白不足 | 端に寄りすぎている | 余白20%程度を確保する |
| 文字視認性 | 小さくて読めない | 太字、大きめ、縁取りを使う |
| サイズ違い | 規定サイズでない | main、tab、01〜40のサイズ確認 |
| 画像切れ | 耳や文字が切れている | 書き出し前に全体確認 |
| 似た画像が多い | 差分が弱い | 表情、ポーズ、小物を変える |

## 内容チェック

| リスク | 内容 | 対策 |
|---|---|---|
| 著作権 | 既存キャラに似ている | あんりぃ独自のデザインを維持 |
| 商標 | ブランド名や作品名 | 使用しない |
| 暴力表現 | 攻撃的な言葉や描写 | かわいい軽口までにする |
| 誤認表現 | 医療・公的機関風 | 誤解される表現を避ける |
| 過激表現 | 性的・差別的・不快表現 | 入れない |
| 強い煽り | 相手を傷つける言葉 | ゆるい表現に変える |
