# あんりぃ LINEスタンプ制作工場

## 目的

2頭身チワワキャラクター「あんりぃ」を使い、LINEスタンプを継続的に量産・販売・分析・シリーズ展開するための制作システム。

10代後半〜40代女性に日常使いされる、感情表現が豊かで何度も使われるスタンプを作り、キャラクターIPとして育てる。

## ターゲット

- 女性
- 友達同士
- カップル
- 家族
- 職場
- 推し活層
- かわいいもの好き

## フォルダ構成

```text
あんりぃ_LINEスタンプ制作工場/
├─ 01_ベース画像/
├─ 02_テーマ別/
├─ 03_プロンプト/
├─ 04_完成画像/
├─ 05_LINE提出用/
├─ 06_タイトル説明文/
├─ 07_販売管理/
├─ 08_人気分析/
└─ 09_リジェクト対策/
```

## 制作フロー

1. `01_ベース画像` に基準キャラ画像を置く
2. `02_テーマ別` でシリーズテーマを決める
3. `03_プロンプト` の画像生成テンプレを使う
4. 9コマ単位で画像生成する
5. Canvaなどで文字入れする
6. 透過PNG化する
7. `04_完成画像` に保管する
8. `05_LINE提出用` に `main.png`、`tab.png`、`01.png〜40.png` を整える
9. `06_タイトル説明文` でタイトルと説明文を作る
10. `09_リジェクト対策` で審査前チェックする
11. 販売後に `08_人気分析` で次シリーズへ反映する

## あんりぃIP固定ルール

- 2頭身チワワ
- ふわふわ毛並み
- 丸目
- パステルカラー
- 日本のかわいい文化
- 女性向け
- やさしく癒される雰囲気
- 表情豊か
- 日常で何度も使える
- シリーズごとに性格を変えすぎない
