# LINEスタンプ プロジェクト

- チワワキャラクター「あんりぃ」
- LINEスタンプ制作
- 40個文言
- 画像生成プロンプト
- Canva文字入れ
- PNG透過
- LINE Creators Market提出
- 販売管理
- 人気分析
- リジェクト対策

## チャット開始時の言い方

```text
今日はLINEスタンプの続きです。
```

## 主なフォルダ

- `あんりぃ_LINEスタンプ制作工場`
- `line_stamp_workflow_template.md`

## 作業ルール

- シリーズごとに新規フォルダまたは新規ファイルで管理する
- 既存の文言や画像案を上書きせず、バージョンを分ける
- 提出用ファイル名は `main.png`、`tab.png`、`01.png〜40.png` に統一する
