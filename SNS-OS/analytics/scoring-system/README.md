# Instagramリール 投稿スコアリングシステム

## 目的

Instagramリールを投稿する前に、バズ期待値を100点満点で数値化する。

「なんとなく良さそう」ではなく、フック、保存率、コメント誘導、視認性などを分解して、投稿前に改善できる状態にする。

## 採点項目

| 項目 | 配点 | 見るポイント |
|---|---:|---|
| フック力 | 12 | 冒頭1〜3秒で止まるか |
| 保存率期待 | 12 | 後で見返したくなるか |
| コメント誘導力 | 10 | コメントしやすい問いがあるか |
| 共感性 | 10 | 視聴者の不安や経験に触れているか |
| 意外性 | 10 | 「え？」と思う要素があるか |
| 専門性 | 10 | 放射線技師視点があるか |
| CTR期待 | 10 | サムネ・タイトルでタップしたくなるか |
| 冒頭3秒維持率 | 10 | 最初の3秒が強い構成か |
| タイトル強度 | 8 | 検索・発見欄で強いか |
| テロップ視認性 | 8 | 小さい画面で読めるか |
| 合計 | 100 |  |

## 出力

- 100点満点
- 改善提案
- 危険ポイント
- バズ確率
- A/Bテスト案

## ファイル構成

```text
scoring-system/
├─ README.md
├─ score_engine.py
├─ rules/
│  └─ scoring_rules.json
├─ templates/
│  └─ 評価テンプレ.md
└─ docs/
   └─ 初心者向け解説.md
```

## 使い方

`score_engine.py` に投稿案の情報を入れて実行すると、スコア、改善提案、危険ポイント、バズ確率、A/Bテスト案が出る。

まずは手動で `templates/評価テンプレ.md` を使い、慣れてきたらPython版を使う。
