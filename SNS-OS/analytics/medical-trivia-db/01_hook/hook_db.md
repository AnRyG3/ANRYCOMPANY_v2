# フックDB

## 伸びやすいフック型

| 型 | 例 | 医療雑学での使い方 |
|---|---|---|
| 誤解破壊 | それ誤解です | 放射線、レントゲン、CTの誤解 |
| 不安代弁 | 放射線って怖いと思ってませんか？ | 一般人の不安に寄り添う |
| 知らない系 | これ9割知らない | 雑学・豆知識 |
| 比較型 | CTとMRI、違いはここ | 保存されやすい |
| 技師視点 | 技師はここを見ています | 専門性・フォロー率 |
| 検査前 | 検査前にこれだけ見て | 保存率狙い |

## そのまま使えるフック

- それ誤解です
- 実は逆です
- 知らないと損
- 放射線って怖いと思ってませんか？
- 現役技師が解説します
- これ9割知らない
- CTが怖い人、これだけ知ってください
- レントゲンで全部見えると思っていませんか？
- MRIとCT、同じだと思っていませんか？
- 検査前にこれだけ保存して
