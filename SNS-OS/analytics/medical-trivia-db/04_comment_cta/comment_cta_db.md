# コメント誘導DB

## コメントされやすい誘導

| 型 | 例 | 狙い |
|---|---|---|
| 経験談 | 検査で不安だったことありますか？ | 一般人の反応 |
| 知らなかった | 知らなかった人はコメントで教えてください | 反応しやすい |
| 比較 | CTとMRIどっちが怖いですか？ | 選択式で答えやすい |
| 医療者向け | 技師さん、これどう説明していますか？ | 専門コメント |
| 施設差 | みなさんの施設ではどうしていますか？ | 医療職の会話 |
