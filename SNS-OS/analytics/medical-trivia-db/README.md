# 医療雑学系ショート動画 構成分析DB

## 目的

Instagram、YouTube Shorts、TikTokで伸びている医療雑学系コンテンツを分析し、再利用できる構成DBとして蓄積する。

分析対象は以下。

- フック
- 尺
- テロップ構成
- コメント誘導
- 保存誘導
- オチ

## 使い方

1. 伸びている医療雑学系投稿を見つける
2. `medical_trivia_structure_db.csv` に1投稿1行で記録する
3. フック、尺、テロップ、CTA、オチを分解する
4. 成功パターンは `07_templates` に型として保存する
5. 次回の「放射線技師の安心ラボ」の台本に流用する

## 分析の基本方針

短尺動画は、冒頭1〜3秒で視聴継続が決まる。医療雑学では、怖がらせるよりも「知らなかった」「あとで見返したい」「家族に教えたい」に着地させる。

## 強い構成

```text
0〜3秒: 誤解・驚き・不安代弁
3〜8秒: 結論
8〜18秒: 理由
18〜28秒: 具体例・比較
28〜35秒: 安心感
最後: 保存・コメント誘導
```

## 参考にする指標

- 再生数
- 保存率
- コメント率
- フォロー率
- 完視聴率
- 平均視聴時間
