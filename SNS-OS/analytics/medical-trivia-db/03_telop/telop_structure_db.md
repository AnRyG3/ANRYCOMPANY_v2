# テロップ構成DB

## 基本ルール

- 1画面最大2行
- 1行12〜16文字程度
- 重要語だけ色を変える
- 冒頭は大文字・太字で止める
- 2〜4秒ごとに画面変化
- 音なしでも意味が分かる

## 伸びやすい構成

```text
1枚目: 強フック
2枚目: 結論
3枚目: 理由1
4枚目: 理由2または比較
5枚目: 技師視点
6枚目: 安心感
7枚目: 保存・コメント誘導
```

## 医療雑学向け例

```text
CTって危険？
結論、必要なら怖がりすぎなくてOK
理由は目的がある検査だから
技師は条件を確認しています
不安なら相談して大丈夫
保存して見返してね
```
