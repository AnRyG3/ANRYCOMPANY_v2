# 特徴量定義

## 冒頭ワード

最初の1〜3秒で表示・発話される言葉。

例:

- CTが怖い人へ
- それ誤解です
- これ9割知らない
- 検査前に見て

## 尺

動画全体の秒数。

分類:

- short: 15秒以下
- standard: 16〜35秒
- education: 36〜60秒
- long: 61秒以上

## テロップ量

動画内に表示される主要テロップ数。

目安:

- 少ない: 1〜4
- 標準: 5〜8
- 多い: 9以上

## 改行数

テロップ内の改行数。

多すぎると読みにくくなるため、1画面2行以内を基本にする。

## 演出

例:

- ズーム
- 青強調
- 赤枠
- 矢印
- 比較表
- 図解
- 顔アップ
- Bロール

## BGMテンポ

分類:

- slow
- medium
- fast
- none

医療系では `medium` または `slow` が基本。

## コメント誘導

コメントを促す問いかけ。

例:

- CTで不安だったことありますか？
- 知らなかった人はコメントで教えてください
- 医療職の方はどう説明していますか？

## 保存誘導

保存を促す文。

例:

- 検査前に見返せるよう保存してね
- 家族に説明する時用に保存
- 忘れないよう保存推奨

## 感情カテゴリ

例:

- 不安→安心
- 驚き
- 共感
- 学び
- 現場感
- 比較納得
- 保存実用
