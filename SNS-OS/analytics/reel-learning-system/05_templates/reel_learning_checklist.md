# 投稿前チェックリスト

- [ ] 冒頭1〜3秒に強いワードがある

- [ ] テロップは5〜8枚程度
- [ ] 1画面2行以内
- [ ] 2〜4秒ごとに画面変化がある
- [ ] BGMがナレーションを邪魔しない
- [ ] コメント誘導がある
- [ ] 保存誘導がある
- [ ] 感情が「不安→安心」または「驚き→納得」になっている
- [ ] 医療不安を煽りすぎていない
