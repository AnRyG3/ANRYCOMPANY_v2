# 初心者向け解説

## このシステムでできること

Instagramリールの投稿結果を入力すると、どの投稿が伸びたか、なぜ伸びたか、次に何を改善すべきかを自動で整理できます。

## まず入力するもの

`csv/post_results.csv` に以下を入れます。

- 投稿ID
- 投稿日
- タイトル
- テーマ
- 冒頭ワード
- 尺
- テロップ量
- 改行数
- 演出
- BGMテンポ
- コメント誘導
- 保存誘導
- 感情カテゴリ
- 再生数
- リーチ
- 保存数
- コメント数
- フォロー数
- 完視聴率

## 自動で計算するもの

- 保存率
- コメント率
- フォロー率
- バズスコア
- 投稿ランキング
- 勝ちパターン
- 失敗原因
- 次回改善案

## バズ判定の考え方

再生数だけではなく、保存率、コメント率、フォロー率、完視聴率を合わせて評価します。

特に教育系リールでは、保存率と完視聴率が重要です。

## 使い方

1. 投稿後の数字を `post_results.csv` に入れる
2. `learning_loop.py` を実行する
3. `reports/latest_learning_report.md` を見る
4. `db/winning_pattern_db.csv` に勝ちパターンが保存される
5. 次回投稿の台本やサムネに反映する

## 見るべきポイント

### 保存率が高い

後で見返したい内容になっています。似たテーマで横展開できます。

### コメント率が高い

問いかけが強いです。コメント誘導の型を再利用します。

### 完視聴率が高い

尺とテンポが合っています。同じ尺で別テーマを作ります。

### フォロー率が高い

専門性やシリーズ化が効いています。次回予告を入れるとさらに伸ばせます。
