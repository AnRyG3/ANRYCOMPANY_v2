# 初心者向け解説

## このパイプラインでできること

ネタを入力すると、AI採点、台本、画像プロンプト、動画編集方針、投稿後の分析、次回改善までを一つの流れで管理できます。

## 最初に見るファイル

- `README.md`
- `config.json`
- `data/ideas.csv`
- `data/kpi_management.csv`
- `outputs/latest_pipeline_report.md`

## 基本の流れ

1. `data/ideas.csv` にネタを追加する
2. `pipeline_manager.py` を実行する
3. `outputs/latest_pipeline_report.md` を確認する
4. 投稿後の数字を `data/kpi_management.csv` に入れる
5. もう一度 `pipeline_manager.py` を実行する
6. `data/buzz_db.csv` に勝ち投稿が保存される

## CSVの使い方

### ideas.csv

投稿前のネタを管理します。

重要な項目:

- theme: 投稿テーマ
- category: 不安解消、医療雑学、整形撮影など
- target: 一般人、若手技師など
- pain_point: 視聴者の不安
- medical_tags: 医療系タグ
- priority: A、B、C

### kpi_management.csv

投稿後の数字を管理します。

重要な項目:

- views: 再生数
- reach: リーチ
- saves: 保存数
- comments: コメント数
- retention_rate: 視聴維持率
- ctr: クリック率

## 点数の見方

- 90点以上: 投稿推奨
- 80点未満: 改善
- 保存率が高い: 保存型として横展開
- コメント率が高い: コメント誘導が強い
- 視聴維持率が高い: 尺と構成が良い
- CTRが高い: サムネとタイトルが強い

## 後から追加しやすい設計

このパイプラインはモジュール化しています。

- 採点を変えたい: `score_engine.py`
- 台本を変えたい: `modules/script_generator.py`
- 画像プロンプトを変えたい: `modules/image_prompt_generator.py`
- 動画設計を変えたい: `modules/video_plan_generator.py`
- 分析を変えたい: `analytics.py`
- 学習を変えたい: `learning_loop.py`
