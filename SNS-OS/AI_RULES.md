# AI RULES

## 基本方針

「放射線技師の安心ラボ」は、医療不安を煽らず、一般の人にわかりやすく、最後は安心感で終わるコンテンツを作る。

## 必須ルール

- 冒頭3秒で興味を引く
- 結論を早めに出す
- 専門用語を減らす
- 一般人の誤解を扱う
- 放射線技師視点を入れる
- 保存したくなる要素を入れる
- コメントしやすい問いを入れる
- 最後は安心感で終わる
- 個別診断に見える表現を避ける
- 強すぎる断定を避ける

## 禁止表現

- 絶対安全です
- 絶対危険です
- 必ず治ります
- これで診断できます
- 医師より正しいです
- 放射線はすべて危険です

## 推奨表現

- 必要な検査なら過度に怖がりすぎなくて大丈夫です
- 不安な時は医師やスタッフに相談してください
- 施設によって運用が異なる場合があります
- 一般的にはこのように考えます
- 放射線技師の視点ではここを確認しています

## 投稿前チェック

- [ ] フックは強いか
- [ ] 結論が遅くないか
- [ ] 保存CTAがあるか
- [ ] コメントCTAがあるか
- [ ] 医療不安を煽りすぎていないか
- [ ] 個別診断に見えないか
- [ ] テロップは読めるか
- [ ] knowledge内の勝ちパターンを参照したか

## AI出力の優先順位

1. 安全性
2. わかりやすさ
3. 保存価値
4. 専門性
5. バズ期待
