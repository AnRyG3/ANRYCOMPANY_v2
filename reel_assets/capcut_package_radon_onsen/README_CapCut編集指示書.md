## おすすめ編集方法

CapCutでは、まず4枚の画像を順番に読み込んでください。
仮動画は完成イメージ確認用です。細かく編集するなら画像4枚から作る方が調整しやすいです。

## ナレーション台本

```text
え、温泉でも放射線を浴びてるって知ってました？

特に関係するのが、ラドン温泉やラジウム温泉です。
ラドンは、地面や岩石の中から出てくる、自然の放射性ガスです。

ラドン温泉では、お湯や空気を通して、ごく微量の放射線を受けることがあります。
でも大事なのは、放射線は「あるかないか」ではなく、「どれくらいの量か」です。

私たちは普段から、空気、食べ物、大地、宇宙からも自然放射線を受けています。
なので、温泉で放射線を浴びているのは本当。
でも、それだけで危険という意味ではありません。

放射線技師目線では、怖がるより、量で考えることが大事です。
ラドン温泉、行ったことありますか？
```

## 画面ごとの補助テロップ

画像には大きいテロップが入っているので、追加テロップは少なめでOKです。

| 0:12-0:24 | ポイントは「量」 |
| 0:24-0:35 | 怖がりすぎなくてOK |

## BGM

- 明るすぎない、落ち着いた教育系BGM
- 音量はナレーションよりかなり小さめ
- 不安を煽る警告音や緊迫系BGMは使わない

## 動き

- 画像ごとに軽いズームインを入れる
- 切り替えはシンプルなフェードでOK
- 3枚目は背景の食べ物・空気・大地の要素を見せたいので、追加テロップを大きく載せすぎない
- 左上のカテゴリラベルは今後の修正版ではなくしてもよい
