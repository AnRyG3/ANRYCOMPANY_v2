# 造影CTリール 100点版

## 狙い
- 目標: 10,000再生を狙う保存型リール
- 想定尺: 40〜48秒
- 視聴者: 検査前に「造影剤が怖い」と検索・不安になる人
- 勝ち筋: 冒頭で不安を代弁し、中盤で「3つだけ覚える」にして保存理由を作る

## 仮想テスト10回の改善結果

| 回 | 想定反応 | 改善ポイント |
|---:|---|---|
| 1 | 冒頭が普通で流される | 「断ったほうがいいですか？」で自分ごと化 |
| 2 | 医療説明が続くと離脱 | 早めに「答えは二択ではない」と結論 |
| 3 | 保存理由が弱い | 「検査前に伝える3つ」を中心に変更 |
| 4 | 怖さを煽りすぎると不安が増える | 危険ワードを抑え、落ち着いた表現に調整 |
| 5 | 安全と言い切ると医療的に危ない | 「必要な場面では診断の助け」に変更 |
| 6 | ぜんそく・腎機能の話が埋もれる | 3項目に分けてテロップ化 |
| 7 | コメントが生まれにくい | 投稿文で「検査前に不安だったこと」を聞く |
| 8 | YouTube Shortsでは締めが弱い | 最後に「なぜ必要？自分の注意点は？」で再提示 |
| 9 | Instagramでは保存導線が欲しい | 投稿文冒頭に「検査前に見返して」を追加 |
| 10 | 全体が長い | 1文を短くし、読み上げテンポを調整 |

## 画面構成

| 秒数 | ナレーション | 画面テロップ |
|---:|---|---|
| 0-2 | 造影剤、断ったほうがいいですか？ | 造影剤、断るべき？ |
| 2-6 | 検査前にそう不安になる人は、少なくありません。 | 不安になるのは自然 |
| 6-12 | でも答えは、危ないから使わない、でも、安心だから何も考えなくていい、でもありません。 | 「危ない」でも<br>「何も考えない」でもない |
| 12-20 | 造影剤は、血管や臓器を見えやすくして、出血、炎症、腫瘍、血管の異常などを確認しやすくする薬です。 | 見えにくい異常を<br>確認しやすくする薬 |
| 20-24 | つまり、必要な場面では診断の助けになります。 | 必要な場面では<br>診断の助けに |
| 24-28 | ただし、検査前に伝えてほしいことが3つあります。 | 検査前に伝えること 3つ |
| 28-34 | 1つ目、過去に造影剤で気分が悪くなったこと。 | 1. 過去の造影剤トラブル |
| 34-39 | 2つ目、ぜんそくや強いアレルギーがあること。 | 2. ぜんそく・強いアレルギー |
| 39-44 | 3つ目、腎臓の機能を指摘されたこと。 | 3. 腎機能を指摘された |
| 44-48 | この3つは、恥ずかしがらずに先に伝えてください。 | 先に伝えてOK |
| 48-55 | 造影剤は、怖がるだけのものではありません。でも、何となく使うものでもありません。 | 怖がりすぎない<br>でも、何となくでもない |
| 55-62 | 大切なのは、なぜ必要なのか、自分に注意点があるかを確認することです。 | なぜ必要？<br>自分の注意点は？ |
