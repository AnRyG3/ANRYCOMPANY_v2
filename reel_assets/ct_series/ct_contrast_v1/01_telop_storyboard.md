# 造影CTリール 台本・テロップ案

## 企画
- テーマ: 造影CT・造影剤の不安をやわらげる
- 想定尺: 35〜45秒
- 目的: 「怖い/安全」の二択ではなく、必要性と注意点を理解してもらう
- トーン: 医療者目線で落ち着いて、断定しすぎない

## 画面構成

| 秒数 | ナレーション | 画面テロップ |
|---:|---|---|
| 0-3 | 造影CTと聞くと、少しこわく感じるかもしれません。 | 造影CT、こわい？ |
| 3-8 | 造影剤は、血管や臓器の様子を見えやすくするために使われる薬です。 | 見えにくい部分を<br>見えやすくする薬 |
| 8-15 | たとえば、出血、炎症、腫瘍、血管の異常を確認したいときに役立ちます。 | 出血・炎症・腫瘍・血管 |
| 15-21 | 一方で、体質や腎臓の状態によって注意が必要な人もいます。 | 注意が必要な人もいます |
| 21-30 | 過去に造影剤で気分が悪くなった人、ぜんそくがある人、腎機能を指摘された人は、事前に伝えることが大切です。 | 事前に伝えたいこと<br>過去の副作用 / ぜんそく / 腎機能 |
| 30-36 | 造影剤は、怖がるだけのものではありません。でも、何となく使うものでもありません。 | 怖がりすぎない<br>でも、何となくでもない |
| 36-43 | 大切なのは、なぜ必要なのか、自分に注意点があるかを確認することです。 | なぜ必要？<br>自分の注意点は？ |

## 冒頭フック候補
- 造影CT、こわい？
- 造影剤って危ないの？
- CTで「造影します」と言われたら

## 画像・動画メモ
- CT装置または検査室のイメージ
- 造影剤ボトルは実物感が出るが、薬剤名が映る場合はぼかす
- 血管や臓器の説明は抽象的な医療イラスト寄り
- 不安を煽る赤一色は避け、白・青緑・薄い黄色で落ち着かせる

個別の判断は医療機関でご相談ください。
