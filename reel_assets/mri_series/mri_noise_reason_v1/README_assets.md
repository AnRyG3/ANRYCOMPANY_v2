## 台本対応

```text
MRI中に一番多い勘違い。
「あの大きな音、壊れてる？」

いいえ。あれは、検査が進んでいる音です。

ガンガン。ドンドン。ビービー。
初めて聞くと、かなり不安になりますよね。

MRIは磁石と電波を使って体の中を見ています。

その磁石が細かく切り替わるたびに、装置が振動して、あの音が出ます。

つまり、音がしている間は「ちゃんと撮影しているサイン」。
音の種類が変わるのも、撮影する部位や方法が変わっているだけです。

だから検査中は「壊れたかも」ではなく、
「今、しっかり撮れてるんだな」と思って大丈夫。

MRIの音は怖い音じゃない。
体の中を一生懸命調べている音です。

検査前の不安が減る情報を発信しています。チャンネル登録しておくと、次の検査のときに役立ちます。

※CTAは `reel_assets/common/reel_end_card_share.png` を使用する。
```
